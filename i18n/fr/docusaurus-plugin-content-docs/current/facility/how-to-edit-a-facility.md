title: Modifier une Installation
---

Suivez ces étapes pour modifier les détails opérationnels d'un lieu physique existant ou pour réaffecter sa hiérarchie parente.

### Étape 1 : Naviguer vers les Installations
Dans la barre latérale, développez **Installations et Actifs > Installations**.

### Étape 2 : Ouvrir le Menu d'Action
Identifiez le lieu spécifique que vous souhaitez modifier. Cliquez sur l'icône **Plus (⋯)** dans la dernière colonne de cette ligne.

### Étape 3 : Sélectionner Modifier
Choisissez **Modifier** dans le menu déroulant. Le formulaire de création original s'ouvrira, pré-rempli avec les données actuelles.

### Étape 4 : Mettre à jour et Enregistrer
Modifiez les champs nécessaires (ex : changer le nom ou déplacer vers une nouvelle installation parente). Cliquez sur **Soumettre** pour enregistrer.
