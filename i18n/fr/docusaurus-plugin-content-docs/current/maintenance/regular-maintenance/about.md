# Maintenance Régulière (Maintenance Préventive)

La **Maintenance Régulière** (PM) vous permet de planifier des **tâches de maintenance récurrentes et automatisées** pour maintenir vos équipements en marche et prévenir les pannes inattendues.

## Pourquoi utiliser la Maintenance Préventive ?

| Maintenance Corrective | Maintenance Préventive |
|---|---|
| Réactive — réparer après la panne | Proactive — prévenir les pannes |
| Temps d'arrêt non planifié | Temps d'arrêt planifié et programmé |
| Coûts de réparation d'urgence élevés | Coûts de maintenance à long terme plus faibles |
| L'équipement tombe en panne de façon imprévisible | Durée de vie de l'équipement prolongée |

## Types de déclencheurs PM

### Option A : Déclencheur Temporel
Le planning PM se déclenche selon un calendrier (ex. : chaque semaine, chaque mois).
- Dans le champ **Déclencheur**, sélectionnez un **Déclencheur Temporel** existant.

:::tip
Les Déclencheurs Temporels doivent être créés dans **Maintenance > Déclencheurs** avant de pouvoir être liés ici.
:::

### Option B : Déclencheur de Compteur
Le planning PM se déclenche en fonction des données d'utilisation d'un Compteur (ex. : toutes les 500 heures de fonctionnement).
- Dans le champ **Déclencheur**, sélectionnez un **Déclencheur de Compteur** existant.

:::tip
Les Déclencheurs de Compteur doivent être créés dans **Maintenance > Déclencheurs** avant de pouvoir être liés ici.
:::
