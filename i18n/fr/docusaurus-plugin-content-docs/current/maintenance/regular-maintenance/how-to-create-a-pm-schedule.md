# Comment créer un planning de Maintenance Régulière (PM)

## Étape 1 : Accéder à la Maintenance Régulière
Depuis la barre latérale, développez **Maintenance** et sélectionnez **Maintenances Régulières**. Cliquez sur **+ Créer**.

## Étape 2 : Définir les paramètres généraux

| Champ | Instruction | Requis |
|---|---|---|
| **Nom** | Saisissez un identifiant clair (ex. : « Compresseur Air — Inspection Mensuelle »). | ✅ Oui |
| **Travailler sur** | Basculez entre Actifs ou Installation et recherchez la cible. | ✅ Oui |
| **Catégorie** | Catégorisez la nature du travail. | ✅ Oui |
| **Priorité** | Sélectionnez le niveau d'urgence. | ✅ Oui |
| **Types** | Spécifiez le type de maintenance exact. | ✅ Oui |
| **Assigner à** | Technicien responsable des tâches récurrentes. | — |
| **Modèle de Procédure** | Joindre une liste de contrôle automatiquement à chaque OT généré. | — |

## Étape 3 : Configurer la source du Déclencheur
Liez un **Déclencheur Temporel** ou un **Déclencheur de Compteur** pour définir quand les Ordres de Travail sont générés automatiquement.

## Étape 4 : Définir la fenêtre de planification *(Optionnel)*

| Champ | Instruction |
|---|---|
| **Date de début** | La date à laquelle le planning PM devient actif. |
| **Date de fin** | La date d'expiration. Laissez vide pour un plan ouvert. |

## Étape 5 : Sauvegarder
Cliquez sur **Soumettre** pour activer le planning de Maintenance Régulière.
