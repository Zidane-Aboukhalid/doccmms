# Comment supprimer un planning de Maintenance Régulière

1. Naviguez vers **Maintenance > Maintenances Régulières**.
2. Localisez le planning. Cliquez sur **Plus (⋯)** dans la colonne de droite.
3. Cliquez sur **Supprimer** (icône corbeille).

:::warning Avant de supprimer
Si vous souhaitez seulement **mettre en pause** le planning, modifiez-le plutôt et définissez une **Date de fin** passée pour arrêter la génération future d'Ordres de Travail sans supprimer l'enregistrement.
:::

:::danger
La suppression d'un planning de Maintenance Régulière est une **action permanente**. Le système arrêtera définitivement de générer des Ordres de Travail depuis ce planning. Les Ordres de Travail déjà générés **ne sont pas affectés**.
:::

4. Confirmez l'action dans la boîte de dialogue de confirmation.
