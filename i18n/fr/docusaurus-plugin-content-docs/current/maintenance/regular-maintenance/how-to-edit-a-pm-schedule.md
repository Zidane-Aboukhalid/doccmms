# Comment modifier un planning de Maintenance Régulière

1. Naviguez vers **Maintenance > Maintenances Régulières**.
2. Localisez le planning. Cliquez sur **Plus (⋯)** et sélectionnez **Modifier**.
3. Mettez à jour les champs nécessaires : Paramètres généraux, Affectation, Source du Déclencheur, Modèle de Procédure, Fenêtre de planification.
4. Cliquez sur **Soumettre** pour appliquer vos modifications.

:::info Note opérationnelle
La modification d'un planning de Maintenance Régulière met à jour le **plan directeur**. Cela affecte tous les Ordres de Travail **futurs** générés depuis ce planning. Il **ne modifie pas rétroactivement** les Ordres de Travail déjà générés ou complétés.
:::
