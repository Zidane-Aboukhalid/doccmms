# Comment supprimer un Ordre de Travail

:::info Contrainte du système
Vous ne pouvez supprimer un Ordre de Travail que si son statut actuel est **Brouillon** ou **Ouvert**. Une fois qu'il progresse au-delà de ces étapes initiales, l'option Supprimer est définitivement verrouillée.
:::

1. Naviguez vers **Maintenance > Ordres de Travail**.
2. Assurez-vous que le statut est **Brouillon** ou **Ouvert**, puis cliquez sur **Plus (⋯)**.
3. Cliquez sur **Supprimer**.

:::danger Avertissement : Pistes d'audit
La suppression d'un Ordre de Travail est une **action permanente**. Si vous devez annuler une tâche mais conserver un enregistrement, utilisez plutôt la fonction **Marquer comme** pour changer le statut en **Annulé**.
:::

4. Confirmez l'action dans la boîte de dialogue de confirmation.
