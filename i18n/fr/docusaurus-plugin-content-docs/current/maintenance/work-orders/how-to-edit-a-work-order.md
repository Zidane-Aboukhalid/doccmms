# Comment modifier un Ordre de Travail

1. Naviguez vers **Maintenance > Ordres de Travail**.
2. Localisez l'Ordre de Travail. Cliquez sur **Plus (⋯)** et sélectionnez **Modifier**.
3. Le formulaire s'ouvre pré-rempli avec les données actuelles. Mettez à jour les champs nécessaires.
4. Cliquez sur **Soumettre** pour sauvegarder l'Ordre de Travail mis à jour.
