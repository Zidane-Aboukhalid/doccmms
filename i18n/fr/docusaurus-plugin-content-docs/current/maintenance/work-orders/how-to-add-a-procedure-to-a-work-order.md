# Comment ajouter une Procédure à un Ordre de Travail

1. Depuis **Maintenance > Ordres de Travail**, cliquez sur le **Code** alphanumérique bleu (ex. : WO12) pour ouvrir le profil détaillé.
2. Dans la navigation interne, cliquez sur **Procédures**, puis sur l'icône **+ (Ajouter)**.
3. Configurez l'affectation de la procédure :

| Champ | Instruction | Requis |
|---|---|---|
| **Modèle de Procédure** | Sélectionnez la liste de contrôle pré-construite à joindre. | ✅ Oui |
| **Assigné à** | Sélectionnez le technicien responsable de compléter cette liste. | — |

:::info Contrainte système : La liste déroulante Assigné
La liste déroulante **Assigné à** est filtrée uniquement pour afficher les techniciens assignés à l'Ordre de Travail parent. Si la liste est vide, ajoutez d'abord des assignés à l'Ordre de Travail principal.
:::

4. Cliquez sur **Soumettre**.
