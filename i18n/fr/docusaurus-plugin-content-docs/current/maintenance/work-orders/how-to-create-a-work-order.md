# Comment créer un Ordre de Travail

## Étape 1 : Accéder aux Ordres de Travail
Depuis la barre latérale, développez **Maintenance** et sélectionnez **Ordres de Travail**. Cliquez sur **+ Créer**.

## Étape 2 : Remplir le formulaire de création

| Champ | Instruction | Requis |
|---|---|---|
| **Nom** | Saisissez un titre clair et concis (ex. : « Remplacement du filtre à air principal »). | ✅ Oui |
| **Statut** | Sélectionnez l'état actuel (Brouillon ou Ouvert). | ✅ Oui |
| **Travailler sur** | Basculez entre Actifs ou Installation, puis recherchez la cible. | ✅ Oui |
| **Catégorie** | Catégorisez la nature du travail (ex. : Correctif, Préventif). | ✅ Oui |
| **Priorité** | Sélectionnez le niveau d'urgence : Aucune, Basse, Moyenne, Haute, ou Critique. | ✅ Oui |
| **Types** | Spécifiez le type de maintenance exact (ex. : Remplacement, Sécurité, Inspection). | ✅ Oui |
| **Assigner à** | Sélectionnez l'utilisateur ou le technicien responsable. | — |
| **Date de début suggérée** | Sélectionnez la date et l'heure de début. | — |
| **Date d'échéance** | Sélectionnez la date limite absolue. | — |
| **Durée estimée** | Saisissez le temps de travail estimé en Heures et Minutes. | — |
| **Description** | Saisissez les détails diagnostiques complets ou les instructions. | — |
| **Compte budgétaire** | Sélectionnez le compte financier applicable. | — |
| **Département de facturation** | Sélectionnez le département responsable du coût. | — |

## Étape 3 : Soumettre l'Ordre de Travail
Vérifiez toutes les données saisies. Cliquez sur **Soumettre** pour créer l'Ordre de Travail et notifier immédiatement le technicien assigné.
