# Comment dupliquer un Ordre de Travail

1. Naviguez vers **Maintenance > Ordres de Travail**.
2. Localisez l'Ordre de Travail à copier. Cliquez sur **Plus (⋯)** dans la colonne de droite.
3. Cliquez sur **Dupliquer**.
4. Un nouveau formulaire s'ouvre pré-rempli avec les données exactes de la tâche originale.

:::warning Première étape cruciale
Vous **devez** mettre à jour la **Date de début suggérée** et la **Date d'échéance**. Si vous laissez les dates historiques, vos métriques de planification seront immédiatement incorrectes.
:::

5. Cliquez sur **Soumettre** pour sauvegarder ce nouvel Ordre de Travail distinct.
