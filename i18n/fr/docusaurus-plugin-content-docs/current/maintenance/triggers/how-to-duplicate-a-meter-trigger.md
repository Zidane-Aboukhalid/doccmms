# Comment dupliquer un Déclencheur de Compteur

1. Naviguez vers **Maintenance > Déclencheurs de Compteur**.
2. Localisez le déclencheur à copier. Cliquez sur l'icône **Plus (⋯)** dans la colonne de droite.
3. Cliquez sur **Dupliquer** dans le menu déroulant.
4. Le formulaire de création s'ouvrira, pré-rempli avec les détails du déclencheur original.

:::note
La fonction de duplication agit comme un clone, copiant toutes les données du déclencheur original. Cela vous permet de maintenir la cohérence sans saisie manuelle.
:::

Cliquez sur **Soumettre** pour sauvegarder le nouveau déclencheur.
