# Comment modifier un Déclencheur de Compteur

1. Naviguez vers **Maintenance > Déclencheurs de Compteur**.
2. Localisez le déclencheur. Cliquez sur **Plus (⋮)** et sélectionnez **Modifier**.
3. Mettez à jour les champs nécessaires :

| Champ | Instruction | Requis |
|---|---|---|
| **Nom** | Saisissez un identifiant descriptif clair. | ✅ Oui |
| **Condition** | Sélectionnez la logique mathématique. Consultez le tableau de logique dans [À propos des Déclencheurs](./about). | ✅ Oui |
| **Seuil** | Saisissez la valeur numérique ou l'intervalle qui active le déclencheur. | ✅ Oui |
| **Est actif** | Cochez pour activer le déclencheur immédiatement. | — |
| **Description** | Fournissez un contexte ou des notes. | — |

4. Cliquez sur **Soumettre** pour sauvegarder. Le système appliquera immédiatement la logique mise à jour à toutes les données entrantes.
