# Comment supprimer un Déclencheur Temporel

1. Naviguez vers **Maintenance > Déclencheur Temporel**.
2. Localisez le déclencheur. Cliquez sur **Plus (⋯)** dans la colonne de droite.
3. Cliquez sur l'option **Supprimer** dans le menu déroulant.

:::warning
La suppression d'un Déclencheur Temporel est une **action permanente** et ne peut pas être annulée. Si vous souhaitez seulement arrêter temporairement l'exécution de la planification, modifiez le déclencheur et **décochez la case Est actif** à la place.
:::

4. Confirmez l'action dans la boîte de dialogue de confirmation.
