# Comment créer un Déclencheur de Compteur

Suivez ces étapes pour configurer une règle de surveillance automatisée pour un compteur existant.

## Étape 1 : Accéder aux Déclencheurs de Compteur
Depuis la barre latérale principale, naviguez vers **Maintenance** et sélectionnez **Déclencheurs de Compteur**.

## Étape 2 : Ouvrir le formulaire de création
Cliquez sur le bouton **+ Créer** situé en haut à droite.

## Étape 3 : Définir les paramètres

| Champ | Instruction | Requis |
|---|---|---|
| **Nom** | Saisissez un identifiant descriptif (ex. : « Alerte chute de pression chaudière »). | ✅ Oui |
| **Compteur** | Sélectionnez le compteur cible dans la liste déroulante. | ✅ Oui |
| **Condition** | Sélectionnez la logique mathématique (ex. : est supérieur à, est inférieur à). | ✅ Oui |
| **Activer le modèle Pic** | *(Conditionnel)* Apparaît uniquement si la condition est **augmente de** ou **diminue de**. | Optionnel |
| **Seuil** | Saisissez la valeur numérique qui active le déclencheur. | ✅ Oui |
| **Est actif** | Cochez cette case pour activer le déclencheur immédiatement. | — |
| **Description** | Fournissez un contexte ou des notes opérationnelles. | — |

## Étape 4 : Finaliser et activer
Cliquez sur **Soumettre** pour sauvegarder la configuration.
