# Comment dupliquer un Déclencheur Temporel

1. Naviguez vers **Maintenance > Déclencheur Temporel**.
2. Localisez le déclencheur. Cliquez sur **Plus (⋯)** dans la colonne de droite.
3. Cliquez sur **Dupliquer** dans le menu déroulant.
4. Le formulaire s'ouvrira pré-rempli avec la configuration exacte du déclencheur original.
   - **Mettez à jour le Nom** immédiatement pour garantir un identifiant unique.
   - Ajustez la Récurrence, les Dates de début/fin ou l'état Actif selon les besoins.
5. Cliquez sur **Soumettre**.

:::note
La duplication copie uniquement la **configuration de la planification**. Elle ne copie **pas** l'historique d'exécution associé au déclencheur original.
:::
