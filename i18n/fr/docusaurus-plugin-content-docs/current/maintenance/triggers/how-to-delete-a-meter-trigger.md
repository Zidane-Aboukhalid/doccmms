# Comment supprimer un Déclencheur de Compteur

1. Naviguez vers **Maintenance > Déclencheurs de Compteur**.
2. Localisez le déclencheur. Cliquez sur l'icône **Plus (⋯)** dans la colonne finale.
3. Sélectionnez l'option **Supprimer** (surlignée en rouge).

:::warning
La suppression d'un déclencheur est **permanente**. Si vous souhaitez seulement arrêter temporairement l'automatisation, décochez plutôt la case **Est actif** dans le formulaire de modification.
:::

4. Confirmez l'action dans la boîte de dialogue de confirmation.
