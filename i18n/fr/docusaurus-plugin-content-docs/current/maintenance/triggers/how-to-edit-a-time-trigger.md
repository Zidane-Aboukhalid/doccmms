# Comment modifier un Déclencheur Temporel

1. Naviguez vers **Maintenance > Déclencheur Temporel**.
2. Localisez le déclencheur. Cliquez sur **Plus (⋯)** et sélectionnez **Modifier**.
3. Mettez à jour les champs :

| Champ | Instruction |
|---|---|
| **Nom** | Mettez à jour l'identifiant textuel. |
| **Est actif** | Cochez ou décochez pour activer/désactiver sans supprimer la planification. |
| **Récurrence** | Modifiez le modèle de planification et l'heure d'exécution exacte. |
| **Date de début** | Ajustez la date à partir de laquelle le déclencheur devient actif. |
| **Date de fin** | Ajustez la date à laquelle le déclencheur expire. |
| **Description** | Mettez à jour le contexte ou les notes opérationnelles. |

4. Cliquez sur **Soumettre** pour appliquer vos modifications.

:::note
Si vous avez modifié les paramètres de **Récurrence**, le système recalcule immédiatement la planification et met à jour la **Prochaine exécution** dans votre tableau de bord.
:::
