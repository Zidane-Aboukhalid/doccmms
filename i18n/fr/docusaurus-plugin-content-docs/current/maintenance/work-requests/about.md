# Demandes de Travail

Une **Demande de Travail** est une soumission formelle pour créer un Ordre de Travail concernant des travaux de maintenance, des réparations ou des problèmes d'installation.

| Type d'utilisateur | Mode de soumission |
|---|---|
| **Utilisateurs internes** | Le personnel avec un accès limité (Demandeurs) crée des Demandes de Travail directement depuis l'application. |
| **Utilisateurs externes** | Les administrateurs peuvent configurer des **Portails Invités** web permettant aux personnes sans compte de soumettre des demandes. |

## Le processus d'approbation

Les responsables de maintenance ou administrateurs doivent examiner toutes les demandes entrantes pour les **approuver** ou les **refuser**.

- **Conversion** : Une fois approuvée, la Demande de Travail est immédiatement convertie en Ordre de Travail actif.

:::tip Quand contourner la Demande de Travail
Seuls les utilisateurs « Demandeur » limités sont tenus d'utiliser ce flux. Les **Administrateurs et les utilisateurs à accès complet** peuvent créer des Ordres de Travail directement.
:::

## Portails Invités

Un **Portail Invité** est un formulaire web autonome permettant à n'importe qui de signaler un problème **sans avoir besoin d'un compte**. Toute personne disposant de l'URL dédiée ou scannant le QR code du portail peut instantanément créer une Demande de Travail.
