# Comment approuver une Demande de Travail (Conversion)

Lorsque vous **approuvez** une Demande de Travail, le système la convertit automatiquement en **Ordre de Travail** actif.

> **Permission requise :** Approuver une Demande de Travail (généralement Responsable Maintenance ou Administrateur)

1. Naviguez vers **Maintenance > Demandes de Travail**.
2. Localisez la demande. Cliquez sur **Plus (⋯)** dans la colonne de droite.
3. Cliquez sur **Approuver**.
4. Le système lance immédiatement le formulaire **Créer un Ordre de Travail**, pré-rempli avec les données de la demande.
   - Vérifiez et complétez tous les champs (Catégorie, Priorité, Types, Assignation, Dates).
5. Cliquez sur **Soumettre**. Le statut de la Demande passera à **Approuvé** et l'Ordre de Travail sera créé.
