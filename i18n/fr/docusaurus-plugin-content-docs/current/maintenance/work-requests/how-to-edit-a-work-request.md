# Comment modifier une Demande de Travail

1. Naviguez vers **Maintenance > Demandes de Travail**.
2. Localisez la Demande de Travail. Cliquez dessus ou sur **Plus (⋯)** et sélectionnez **Modifier**.
3. Mettez à jour les champs pertinents : **Nom**, **Priorité**, **Actif**, **Description**, **Note**.
4. Cliquez sur **Soumettre** pour sauvegarder la Demande de Travail mise à jour.
