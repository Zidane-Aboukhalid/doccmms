# Comment supprimer une Demande de Travail

> **Permission requise :** Supprimer une Demande de Travail (généralement Administrateur)

:::warning
Avant de supprimer, demandez-vous si vous avez besoin de conserver un enregistrement historique. Si vous souhaitez simplement la retirer de votre file d'attente, la **Refuser** (ce qui conserve l'enregistrement) est plus approprié que la supprimer.
:::

1. Naviguez vers **Maintenance > Demandes de Travail**.
2. Cliquez sur **Plus (⋯)** dans la colonne de droite.
3. Cliquez sur **Supprimer** (icône corbeille).
4. Confirmez l'action dans la boîte de dialogue.
