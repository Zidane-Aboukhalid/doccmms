# Comment créer une Demande de Travail

## Étape 1 : Accéder aux Demandes de Travail
Depuis la barre latérale, développez **Maintenance** et sélectionnez **Demandes de Travail**. Cliquez sur **+ Créer**.

## Étape 2 : Détailler le problème

| Champ | Instruction | Requis |
|---|---|---|
| **Nom** | Saisissez un titre clair et bref (ex. : « Vibration forte et fuite d'huile sur la ligne principale »). | ✅ Oui |
| **Priorité** | Sélectionnez le niveau d'urgence perçu. | ✅ Oui |
| **Actif** | Recherchez et sélectionnez l'équipement défaillant. | — |
| **Description** | Fournissez les détails diagnostiques complets. | — |
| **Note** | Ajoutez un contexte administratif ou des instructions pour le réviseur. | — |

:::tip Bonne pratique : Toujours étiqueter l'Actif
Même si le champ Actif est techniquement optionnel, vous devriez **toujours** tenter de lier votre demande à une machine spécifique. Sinon, le responsable devra physiquement localiser le problème, ce qui **retarde le processus d'approbation et de réparation**.
:::

## Étape 3 : Soumettre la Demande de Travail
Vérifiez vos informations. Cliquez sur **Soumettre** pour envoyer la demande dans la file d'attente du responsable.
