# Comment refuser une Demande de Travail

> **Permission requise :** Refuser une Demande de Travail (généralement Responsable Maintenance ou Administrateur)

1. Naviguez vers **Maintenance > Demandes de Travail**.
2. Cliquez sur **Plus (⋯)** dans la colonne de droite de la demande spécifique.
3. Cliquez sur **Refuser** dans le menu déroulant.
4. **Fournir un motif** *(si demandé)* : Le système peut vous inviter à saisir un motif de refus, visible par le demandeur.
5. Confirmez l'action. Le statut passera à **Refusé** et le demandeur sera notifié.
