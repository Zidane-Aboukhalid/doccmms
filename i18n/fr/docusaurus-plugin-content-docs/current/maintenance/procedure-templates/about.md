# Modèles de Procédures

Les Modèles de Procédures vous permettent de créer des **Procédures Opérationnelles Standard (POS)**, des listes de contrôle de sécurité ou des flux de travail de maintenance qui peuvent être joints aux Ordres de Travail. Ils guident les techniciens étape par étape et capturent des données structurées.

## Guide de référence : Types de saisie des tâches

| Type de saisie | Description | Meilleur cas d'usage |
|---|---|---|
| **Champ texte** | Permet au technicien de saisir une réponse textuelle courte. | Enregistrer des numéros de série, des noms de pièces ou des notes d'observation. |
| **Case à cocher** | Boîte de basculement simple pour la confirmation d'un seul élément. | Vérifications visuelles de base (ex. : « Protections en place », « Zone nettoyée »). |
| **Champ numérique** | Limite strictement la saisie aux valeurs numériques. | Enregistrer des relevés, températures, PSI ou niveaux de tension. |
| **Choix unique** | Présente une liste personnalisée d'options, forçant le technicien à sélectionner exactement une réponse. | Conditions de statut (ex. : « Réussi / Nécessite réparation / Échoué »). |
| **Liste de contrôle** | Présente une liste personnalisée d'options, permettant de sélectionner plusieurs réponses. | Contrôles multi-parties (ex. : « Sélectionner tous les fluides complétés : Huile, Liquide de refroidissement »). |
| **Contrôle d'inspection** | Format de saisie de condition standardisé conçu pour les inspections d'actifs. | Audits périodiques de l'état physique des actifs. |
| **Oui-Non-N/A** | Fournit trois boutons de sélection rapide préétablis. | Questions de conformité ou de sécurité standard. |
| **Date** | Ouvre un sélecteur de calendrier pour enregistrer un jour spécifique. | Enregistrer des dates d'expiration ou d'installation de pièces. |
| **Date et heure** | Ouvre un sélecteur de calendrier et d'horloge. | Enregistrer le moment précis d'un incident ou d'un test. |
