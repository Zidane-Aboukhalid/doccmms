# Comment modifier un Modèle de Procédure

1. Naviguez vers **Maintenance > Modèles de Procédures**.
2. Localisez la procédure. Cliquez sur **Plus (⋯)** et sélectionnez **Modifier**.
3. Le constructeur s'ouvre en mode **ÉDITEUR** pré-rempli avec les données actuelles. Vous pouvez mettre à jour : **Nom**, **Actif**, **Types**, **Description**, les blocs existants (texte, type de saisie, bascule requis, règles logiques).
4. Basculez en **APERÇU** pour vérifier, puis cliquez sur **Soumettre**.

:::info Note opérationnelle
La mise à jour du Modèle de Procédure modifie le **plan directeur**. Ce changement s'applique aux ordres de travail **futurs** utilisant ce modèle, mais **ne modifie pas rétroactivement** les ordres de travail déjà complétés.
:::
