# Comment dupliquer un Modèle de Procédure

1. Naviguez vers **Maintenance > Modèles de Procédures**.
2. Localisez le modèle. Cliquez sur **Plus (⋯)** dans la colonne de droite.
3. Cliquez sur **Dupliquer**.
4. Le constructeur s'ouvre avec une **copie exacte** de la configuration originale.

:::warning Première étape cruciale
**Mettez à jour immédiatement le champ Nom**. Si vous le laissez identique à l'original, les techniciens ne sauront pas quelle POS sélectionner.
:::

5. Ajustez les paramètres, tâches ou logique selon les besoins.
6. Vérifiez en **APERÇU**, puis cliquez sur **Soumettre**.
