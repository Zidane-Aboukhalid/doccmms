# Comment supprimer un Modèle de Procédure

1. Naviguez vers **Maintenance > Modèles de Procédures**.
2. Localisez la procédure. Cliquez sur **Plus (⋯)** dans la colonne de droite.
3. Cliquez sur l'option **Supprimer** (icône corbeille).

:::danger Avertissement critique
La suppression d'un Modèle de Procédure est une **action permanente**.
- **Impact futur** : Ce modèle ne sera plus disponible pour être joint à de nouveaux Ordres de Travail.
- **Données historiques** : La suppression du modèle principal **n'efface pas** les données des Ordres de Travail déjà complétés avec ce modèle.
:::

4. Confirmez l'action dans la boîte de dialogue de confirmation.
