## Comment enregistrer un relevé de Compteur

> **Permission requise :** Gérer les relevés de Compteur

Il existe **deux méthodes principales** pour saisir manuellement un nouveau relevé pour un compteur existant.

### Étape 1 : Accéder au formulaire de saisie

**Méthode 1 : Ajout rapide via la vue liste**
1. Naviguez vers **Maintenance > Compteurs**.
2. Localisez le compteur spécifique dans la liste.
3. Cliquez sur l'icône **Relevé** associée à la ligne de ce compteur.

**Méthode 2 : Ajout via la vue détaillée**
1. Naviguez vers **Maintenance > Compteurs**.
2. Cliquez sur le compteur cible pour ouvrir sa vue détaillée.
3. Cliquez sur le bouton **Relevé** situé dans l'interface.

### Étape 2 : Enregistrer les données
Remplissez le formulaire de saisie. Le système évaluera immédiatement les nouvelles données par rapport à tous les Déclencheurs de Compteur actifs.

| Champ | Instruction | Requis |
|---|---|---|
| **Compteur** | Le système prédéfinit ce champ selon votre sélection. Il ne peut pas être modifié. | ✅ Système |
| **Valeur de relevé** | Saisissez les données numériques actuelles de l'équipement. | ✅ Oui |
| **Note** | Consignez toute observation visuelle ou contexte concernant le relevé. | — |

### Étape 3 : Confirmer la saisie
Cliquez sur le bouton **Soumettre** pour ajouter le nouveau relevé au journal historique du compteur.

:::note
La modification des relevés enregistrés nécessite la permission **Autoriser la modification rétroactive de l'historique**.
:::
