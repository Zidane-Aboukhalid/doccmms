## Comment créer un Compteur

> **Permission requise :** Ajouter un Compteur

Suivez ces étapes pour configurer un nouveau compteur dans le système.

### Étape 1 : Accéder aux Compteurs
Depuis la barre latérale principale, naviguez vers **Maintenance** et sélectionnez le sous-menu **Compteurs**.

### Étape 2 : Ouvrir le formulaire de création
Cliquez sur le bouton **Créer** situé dans la barre d'actions supérieure de la liste des compteurs.

### Étape 3 : Définir les paramètres du compteur
Remplissez les champs du formulaire de création :

| Champ | Instruction | Requis |
|---|---|---|
| **Nom** | Saisissez un identifiant unique (ex. : « Compresseur Air 01 Heures »). | ✅ Oui |
| **Actif** | Sélectionnez l'équipement spécifique auquel ce compteur appartient. | — |
| **Unité de mesure** | Définissez l'unité (ex. : Degré Celsius, Litre, Volt, etc.). | ✅ Oui |
| **Valeur de relevé** | Saisissez l'état actuel ou initial du compteur si l'actif est déjà en service. | ✅ Oui |
| **Description** | Fournissez un contexte supplémentaire sur l'objectif ou l'emplacement du compteur. | — |

### Étape 4 : Sauvegarder
Cliquez sur le bouton **Soumettre** pour finaliser la configuration du compteur.
