## Comment supprimer un Compteur

> **Permission requise :** Supprimer un Compteur

La suppression d'un compteur le **retire définitivement** du système.

:::danger Avertissement
La suppression d'un compteur est une **action destructive**. Elle peut effacer définitivement toutes les données de relevés historiques associées et désactiver tous les Déclencheurs de Compteur actifs ou les planifications de Maintenance Préventive qui dépendent de ses données.
:::

### Étape 1 : Localiser le compteur
Naviguez vers **Maintenance > Compteurs**. Utilisez la barre de recherche ou les filtres pour trouver le compteur à supprimer.

### Étape 2 : Initier la suppression
Sélectionnez le compteur dans la liste. Cliquez sur le bouton **Supprimer** situé dans la colonne d'actions ou dans la vue détaillée du compteur.

### Étape 3 : Confirmer la suppression
Une invite de confirmation apparaît pour éviter la perte accidentelle de données. Examinez l'avertissement et cliquez sur **Supprimer** pour retirer définitivement le compteur et ses données associées du système.
