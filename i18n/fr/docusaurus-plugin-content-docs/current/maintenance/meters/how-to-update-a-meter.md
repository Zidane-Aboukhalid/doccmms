## Comment modifier un Compteur

> **Permission requise :** Modifier un Compteur

Suivez ces étapes pour modifier la configuration d'un compteur existant.

### Étape 1 : Localiser le compteur
Naviguez vers **Maintenance > Compteurs**. Utilisez la barre de recherche ou les filtres pour trouver le compteur à modifier.

### Étape 2 : Accéder au formulaire de modification
Sélectionnez un compteur dans la liste. Cliquez sur le bouton d'options (**⋯**) et sélectionnez **Modifier**. Vous pouvez également trouver le bouton Modifier en haut à droite de la page de détails du compteur.

### Étape 3 : Modifier les paramètres

| Champ | Instruction | Requis |
|---|---|---|
| **Nom** | Mettez à jour l'identifiant du compteur. | ✅ Oui |
| **Actif** | Réaffectez le compteur à un actif différent si nécessaire. | — |
| **Unité de mesure** | Définissez l'unité (ex. : Degré Celsius, Litre, Volt, etc.). | ✅ Oui |
| **Description** | Mettez à jour les détails contextuels du compteur. | — |

:::tip Activer les mises à jour d'unité de compteur
Pour activer les mises à jour d'unité, accédez à **Portail Admin > Paramètres du locataire > Permissions opérationnelles** et sélectionnez **Autoriser la mise à jour de l'unité du compteur**. Le champ Unité de mesure deviendra alors modifiable.
:::

### Étape 4 : Finaliser les modifications
Cliquez sur le bouton **Soumettre** pour sauvegarder la configuration mise à jour.
