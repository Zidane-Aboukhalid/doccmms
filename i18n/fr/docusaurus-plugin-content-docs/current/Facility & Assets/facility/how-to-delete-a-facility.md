title: Supprimer une Installation
---

Suivez ces étapes pour retirer définitivement un lieu physique ou une zone opérationnelle de la base de données.

### Étape 1 : Naviguer vers les Installations
Dans la barre latérale, développez **Installations et Actifs > Installations**.

### Étape 2 : Ouvrir le Menu d'Action
Identifiez l'installation à supprimer. Cliquez sur l'icône **Plus (⋯)** dans la dernière colonne.

### Étape 3 : Sélectionner Supprimer
Choisissez l'option **Supprimer** (icône poubelle).

:::warning Avertissement Système
La suppression est définitive. Le système bloquera cette action si l'installation sert actuellement de lieu de stockage et contient un inventaire actif. Vous devez déplacer ou retirer toutes les pièces et fournitures avant de pouvoir supprimer l'installation.
:::

### Étape 4 : Confirmer la Suppression
Vérifiez le nom et confirmez la suppression définitive.
