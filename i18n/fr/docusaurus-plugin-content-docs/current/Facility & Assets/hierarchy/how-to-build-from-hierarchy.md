title: Construire depuis la Hiérarchie
---

Vous n'avez pas besoin de quitter l'arbre pour étendre votre infrastructure. Vous pouvez générer de nouveaux enregistrements directement depuis le tableau de bord de la Hiérarchie.

### Étape 1 : Naviguer vers la Hiérarchie
Depuis la barre latérale principale, développez le menu **Installations et Actifs** et sélectionnez **Hiérarchie**.

### Étape 2 : Ouvrir le Menu de Création
Cliquez sur le bouton noir **+ Créer** situé dans le coin supérieur droit du tableau de bord.

### Étape 3 : Sélectionner le Type d'Enregistrement
Un menu déroulant apparaîtra. Sélectionnez soit **Actif**, soit **Installation** selon ce que vous devez ajouter à la base de données.

:::info Note de Navigation
La sélection d'une option dans ce menu déclenchera les mêmes formulaires de création que ceux utilisés dans tout le système.
*   Pour voir les champs requis pour ajouter un lieu, consultez la section [Créer une Installation](/docs/facility/how-to-create-a-facility).
*   Pour voir les champs requis pour ajouter un équipement, consultez la section [Créer un Actif](/docs/assets/how-to-create-an-asset).
:::
