# Installations et Actifs

Le module **Installations et Actifs** est l'endroit où vous gérez tout ce que vous entreprenez de maintenir. Des grands emplacements physiques aux équipements et outils granulaires, ce module conserve un enregistrement numérique complet de votre infrastructure opérationnelle.

---

## Sous-modules

Sélectionnez une section pour gérer vos actifs :

*   [**Emplacements**](/docs/assets/locations/about) — Configurez la hiérarchie de votre installation (Sites, Bâtiments, Pièces).
*   [**Équipements**](/docs/assets/equipment/about) — Suivez les machines individuelles, les types de machines et l'historique de maintenance.
*   [**Outils**](/docs/assets/tools/about) — Gérez les actifs partagés ou les équipements portables qui nécessitent un suivi.
*   [**Consommation de Pièces**](/docs/parts-and-supplies/about) — *Remarque : Les pièces sont gérées séparément mais peuvent être liées à n'importe quel actif.*

---

## Pourquoi suivre les actifs ?

En configurant correctement vos actifs, vous pouvez :
1.  **Suivre les coûts** : Voir exactement combien la maintenance dépense pour des machines spécifiques.
2.  **Surveiller la santé** : Afficher les relevés de compteurs et l'historique des déclencheurs.
3.  **Historique d'audit** : Conservez un enregistrement complet de tous les bons de travail effectués sur un actif.
