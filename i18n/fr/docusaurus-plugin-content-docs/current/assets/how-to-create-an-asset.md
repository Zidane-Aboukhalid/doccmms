title: Créer un Actif
---

La création d'un profil d'actif précis est une étape fondamentale pour le suivi des registres de maintenance, le calcul des coûts d'indisponibilité et la gestion du cycle de vie des équipements.

### Étape 1 : Naviguer vers le Tableau de Bord des Actifs
Depuis la barre latérale principale, développez **Installations et Actifs** et sélectionnez **Actifs**. Cliquez sur le bouton noir **+ Créer** situé dans le coin supérieur droit.

### Étape 2 : Définir les Paramètres de l'Actif
Remplissez le formulaire de création. Les champs marqués d'un astérisque rouge sont obligatoires.

| Champ | Instruction | Requis |
| :--- | :--- | :--- |
| **Nom** | Entrez un identifiant unique (ex : "Compresseur 01"). | Oui |
| **Statut** | Sélectionnez l'état opérationnel (ex : Actif, En attente). | Oui |
| **Raison** | Spécifiez le motif du statut choisi. | Oui |
| **Parent** | Basculez entre **Installation** (autonome) ou **Actif** (sous-composant). | Non |
| **Criticité** | Définissez l'importance : Faible, Moyenne ou Haute. | Non |
| **Types** | Catégorisez l'actif (ex : CVC, Convoyeur). | Non |
| **Équipes** | Assignez l'équipe de maintenance responsable. | Non |
| **Code-barres** | Entrez ou scannez un code pour une identification rapide. | Non |
| **Coût d'arrêt** | Perte financière estimée par heure d'arrêt. | Non |
| **Date d'achat** | Sélectionnez la date d'acquisition. | Non |
| **Coût d'achat** | Coût financier initial de l'acquisition. | Non |

### Étape 3 : Enregistrer l'Actif
Vérifiez tous les paramètres et cliquez sur **Soumettre** en bas à droite du formulaire.
