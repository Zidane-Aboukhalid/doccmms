---

## Vermögenswert bearbeiten
Sie können ein Asset-Profil jederzeit aktualisieren, um die Datengenauigkeit zu gewährleisten.

1.  Navigieren Sie zu **Anlagen und Vermögenswerte > Vermögenswerte**.
2.  Klicken Sie auf das Symbol **Mehr (⋯)** ganz rechts in der Asset-Zeile.
3.  Wählen Sie **Bearbeiten**.
4.  Ändern Sie die erforderlichen Felder und klicken Sie auf **Senden**.

---

## Vermögenswert löschen
Befolgen Sie diese Schritte, um ein Geräteprofil dauerhaft zu entfernen.

1.  Navigieren Sie zu **Anlagen und Vermögenswerte > Vermögenswerte**.
2.  Klicken Sie auf das Symbol **Mehr (⋯)** und wählen Sie **Löschen** (Papierkorb-Symbol).

:::warning Systemwarnung
Das Löschen ist dauerhaft. Das System blockiert das Löschen, wenn das Asset mit vorhandenen Datensätzen (z. B. Arbeitsaufträgen) verknüpft ist. Erwägen Sie stattdessen, das Asset als **Ausgemustert** zu markieren.
:::

3.  Bestätigen Sie das Löschen in der Eingabeaufforderung.
