title: Anlage erstellen
---

Befolgen Sie diese Schritte, um einen neuen physischen Standort, ein Gebäude oder eine Betriebszone in der Systemdatenbank zu registrieren.

### Schritt 1: Navigieren zu Anlagen
Erweitern Sie im Hauptseitenmenü **Anlagen und Vermögenswerte** und wählen Sie **Anlagen**. Klicken Sie auf die schwarze Schaltfläche **+ Erstellen** in der oberen rechten Ecke des Dashboards.

### Schritt 2: Anlagenparameter definieren
Füllen Sie das Erstellungsformular aus, um die Details des Standorts zu erfassen und seinen betrieblichen Zweck festzulegen.
(Hinweis: Mit einem roten Sternchen gekennzeichnete Felder sind Pflichtfelder).

| Feld | Anleitung | Erforderlich |
| :--- | :--- | :--- |
| **Name** | Geben Sie eine klare, eindeutige Kennung ein (z. B. „Hauptlager“). | Ja |
| **Übergeordnete Anlage** | Erstellen Sie eine räumliche Hierarchie, indem Sie einen übergeordneten Standort auswählen. | Nein |
| **Ist ein Lager** | Aktivieren Sie dieses Kontrollkästchen, um den Standort als Lagerzentrum zu nutzen. | Nein |
| **Erstellungsdatum** | Wählen Sie das Datum der Errichtung oder Inbetriebnahme der Anlage. | Nein |
| **Adresse** | Geben Sie die physische Adresse oder den geografischen Standort ein. | Nein |
| **Beschreibung** | Geben Sie allgemeine Notizen oder Zugangsanweisungen an. | Nein |

:::info Systemlogik
Die Aktivierung der Option **Ist ein Lager** ermöglicht der Anlage, Ersatzteile, Vorräte und Vermögenswerte physisch zu lagern und zu verwalten.
:::

### Schritt 3: Anlage speichern
Überprüfen Sie die eingegebenen Parameter und klicken Sie auf **Senden**, um die Anlage dauerhaft zu registrieren.
