title: Aus der Hierarchie aufbauen
---

Sie müssen den Baum nicht verlassen, um Ihre Infrastruktur zu erweitern. Sie können neue Datensätze direkt aus dem Hierarchie-Dashboard generieren.

### Schritt 1: Navigieren zur Hierarchie
Erweitern Sie im Hauptseitenmenü das Menü **Anlagen und Vermögenswerte** und wählen Sie **Hierarchie**.

### Schritt 2: Erstellungsmenü öffnen
Klicken Sie auf die schwarze Schaltfläche **+ Erstellen** in der oberen rechten Ecke des Dashboards.

### Schritt 3: Datensatztyp auswählen
Ein Dropdown-Menü wird angezeigt. Wählen Sie entweder **Vermögenswert** oder **Anlage**, je nachdem, was Sie hinzufügen möchten.

:::info Navigationshinweis
Durch Auswahl einer Option aus diesem Menü werden dieselben Erstellungsformulare aufgerufen, die im gesamten System verwendet werden.
*   Die erforderlichen Felder zum Hinzufügen eines Standorts finden Sie im Abschnitt [Anlage erstellen](/docs/facility/how-to-create-a-facility).
*   Die erforderlichen Felder zum Hinzufügen von Ausrüstung finden Sie im Abschnitt [Vermögenswert erstellen](/docs/assets/how-to-create-an-asset).
:::
