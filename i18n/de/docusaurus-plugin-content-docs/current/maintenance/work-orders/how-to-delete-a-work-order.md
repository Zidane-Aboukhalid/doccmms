# Wie man einen Arbeitsauftrag löscht

:::info Systemeinschränkung
Sie können einen Arbeitsauftrag nur löschen, wenn sein aktueller Status **Entwurf** oder **Offen** ist. Sobald er über diese anfänglichen Phasen hinausgeht, wird die Löschoption dauerhaft gesperrt.
:::

1. Navigieren Sie zu **Wartung > Arbeitsaufträge**.
2. Stellen Sie sicher, dass der Status **Entwurf** oder **Offen** ist, dann klicken Sie auf **Mehr (⋯)**.
3. Klicken Sie auf **Löschen**.

:::danger Warnung: Prüfpfade
Wenn Sie eine Aufgabe stornieren müssen, aber einen Datensatz behalten möchten, verwenden Sie stattdessen die Funktion **Als markieren**, um den Status auf **Storniert** zu ändern.
:::

4. Bestätigen Sie die Aktion im Bestätigungsdialog.
