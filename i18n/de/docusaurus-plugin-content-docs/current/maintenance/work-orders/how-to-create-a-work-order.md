# Wie man einen Arbeitsauftrag erstellt

## Schritt 1: Zu Arbeitsaufträgen navigieren
Erweitern Sie **Wartung** in der Seitenleiste und wählen Sie **Arbeitsaufträge**. Klicken Sie auf **+ Erstellen**.

## Schritt 2: Erstellungsformular ausfüllen

| Feld | Anweisung | Pflichtfeld |
|---|---|---|
| **Name** | Geben Sie einen klaren, prägnanten Titel ein (z. B. „Hauptluftfilter ersetzen"). | ✅ Ja |
| **Status** | Wählen Sie den aktuellen Status (Entwurf oder Offen). | ✅ Ja |
| **Arbeiten an** | Wechseln Sie zwischen Assets oder Anlage, dann suchen Sie das Ziel. | ✅ Ja |
| **Kategorie** | Kategorisieren Sie die Art der Arbeit (z. B. Korrektiv, Präventiv). | ✅ Ja |
| **Priorität** | Wählen Sie die Dringlichkeitsstufe: Keine, Niedrig, Mittel, Hoch oder Kritisch. | ✅ Ja |
| **Typen** | Geben Sie den genauen Wartungstyp an (z. B. Austausch, Sicherheit, Inspektion). | ✅ Ja |
| **Zuweisen an** | Wählen Sie den verantwortlichen Benutzer oder Techniker. | — |
| **Vorgeschlagenes Startdatum** | Wählen Sie Startdatum und -uhrzeit. | — |
| **Fälligkeitsdatum** | Wählen Sie die absolute Frist. | — |
| **Geschätzte Dauer** | Geben Sie die geschätzte Arbeitszeit in Stunden und Minuten ein. | — |
| **Beschreibung** | Geben Sie vollständige Diagnosedetails oder Anweisungen ein. | — |
| **Budgetkonto** | Wählen Sie das zutreffende Finanzkonto. | — |
| **Kostenstelle** | Wählen Sie die verantwortliche interne Abteilung. | — |

## Schritt 3: Arbeitsauftrag einreichen
Klicken Sie auf **Einreichen**, um den Arbeitsauftrag zu erstellen und den zugewiesenen Techniker sofort zu benachrichtigen.
