# Wie man ein Verfahren zu einem Arbeitsauftrag hinzufügt

1. Klicken Sie unter **Wartung > Arbeitsaufträge** auf den blauen alphanumerischen **Code** (z. B. WO12), um das Detailprofil zu öffnen.
2. Klicken Sie in der internen Navigation auf **Verfahren**, dann auf das **+ (Hinzufügen)**-Symbol.
3. Konfigurieren Sie die Verfahrenszuweisung:

| Feld | Anweisung | Pflichtfeld |
|---|---|---|
| **Verfahrensvorlage** | Wählen Sie die vorgefertigte Checkliste, die angehängt werden soll. | ✅ Ja |
| **Zugewiesen an** | Wählen Sie den Techniker, der für das Abschließen dieser Liste verantwortlich ist. | — |

:::info Systemeinschränkung: Dropdown Zugewiesen an
Das Dropdown **Zugewiesen an** ist gefiltert, um nur Techniker anzuzeigen, die dem übergeordneten Arbeitsauftrag zugewiesen sind. Wenn es leer ist, fügen Sie zuerst Mitarbeiter zum Hauptarbeitsauftrag hinzu.
:::

4. Klicken Sie auf **Einreichen**.
