# Wie man einen Arbeitsauftrag dupliziert

1. Navigieren Sie zu **Wartung > Arbeitsaufträge**.
2. Lokalisieren Sie den Arbeitsauftrag. Klicken Sie auf **Mehr (⋯)** und dann auf **Duplizieren**.
3. Ein neues Formular öffnet sich, vorausgefüllt mit den exakten Daten der ursprünglichen Aufgabe.

:::warning Entscheidender erster Schritt
Sie **müssen** das **Vorgeschlagene Startdatum** und das **Fälligkeitsdatum** aktualisieren. Wenn Sie historische Daten belassen, werden Ihre Planungsmetriken sofort ungenau.
:::

4. Klicken Sie auf **Einreichen**, um diesen neuen, eigenständigen Arbeitsauftrag zu speichern.
