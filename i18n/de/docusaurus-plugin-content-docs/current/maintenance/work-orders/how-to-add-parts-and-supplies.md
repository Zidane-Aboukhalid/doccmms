# Wie man Teile und Zubehör zu einem Arbeitsauftrag hinzufügt

1. Öffnen Sie das Detailprofil des Arbeitsauftrags, indem Sie auf seinen blauen **Code** klicken.
2. Wählen Sie in der internen Navigation **Teile und Zubehör**, dann klicken Sie auf **+ (Hinzufügen)**.

## Methode A: Teile-Kit verwenden
1. Klicken Sie auf **Teile-Kit auswählen**.
2. Wählen Sie Ihr Kit. Das System füllt automatisch Artikel und Mengen aus.
3. Wählen Sie den spezifischen **Standort** für jeden Artikel.

## Methode B: Einzelne Artikel hinzufügen
1. Klicken Sie auf **Neuer Artikel**.
2. Konfigurieren Sie die drei Pflichtfelder:
   - **Teil**: Suchen und wählen Sie den Artikel aus.
   - **Standort**: Wählen Sie den physischen Lagerort.
   - **Menge**: Geben Sie die genaue Anzahl verbrauchter Einheiten ein.

:::tip Teil aus mehreren Standorten beziehen
Fügen Sie dasselbe Teil mehrfach mit verschiedenen Lagerorten zur Liste hinzu.

**Beispiel:** Sie benötigen 4 „Schwerlast-V-Riemen". Hauptlager hat 3, Reservelager hat 1:
- **Zeile 1**: Teil: V-Riemen | Standort: Hauptlager | Menge: 3
- **Zeile 2**: Teil: V-Riemen | Standort: Reservelager | Menge: 1
:::

3. Klicken Sie auf **Einreichen**, um diese Mengen von Ihrem aktiven Inventar abzuziehen.
