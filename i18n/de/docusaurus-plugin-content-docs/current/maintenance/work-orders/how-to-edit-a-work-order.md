# Wie man einen Arbeitsauftrag bearbeitet

1. Navigieren Sie zu **Wartung > Arbeitsaufträge**.
2. Lokalisieren Sie den Arbeitsauftrag. Klicken Sie auf **Mehr (⋯)** und wählen Sie **Bearbeiten**.
3. Das Formular öffnet sich vorausgefüllt mit aktuellen Daten. Aktualisieren Sie die erforderlichen Felder.
4. Klicken Sie auf **Einreichen**, um den aktualisierten Arbeitsauftrag zu speichern.
