# Wie man einen Zeitauslöser bearbeitet

1. Navigieren Sie zu **Wartung > Zeitauslöser**.
2. Lokalisieren Sie den Auslöser. Klicken Sie auf **Mehr (⋯)** und wählen Sie **Bearbeiten**.
3. Aktualisieren Sie die Felder:

| Feld | Anweisung |
|---|---|
| **Name** | Aktualisieren Sie die Textkennung. |
| **Ist aktiv** | Aktivieren oder deaktivieren Sie, ohne den Zeitplan zu löschen. |
| **Wiederholung** | Ändern Sie das Planungsmuster und die genaue Ausführungszeit. |
| **Startdatum** | Passen Sie das Datum an, ab dem der Auslöser aktiv wird. |
| **Enddatum** | Passen Sie das Ablaufdatum des Auslösers an. |
| **Beschreibung** | Aktualisieren Sie den Kontext oder Betriebsnotizen. |

4. Klicken Sie auf **Einreichen**, um Ihre Änderungen anzuwenden.
