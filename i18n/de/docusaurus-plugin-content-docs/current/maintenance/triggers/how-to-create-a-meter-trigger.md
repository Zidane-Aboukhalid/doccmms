# Wie man einen Zählerauslöser erstellt

## Schritt 1: Zu Zählerauslösern navigieren
Gehen Sie in der Seitenleiste zu **Wartung** und wählen Sie **Zählerauslöser**.

## Schritt 2: Erstellungsformular öffnen
Klicken Sie auf **+ Erstellen** oben rechts.

## Schritt 3: Parameter definieren

| Feld | Anweisung | Pflichtfeld |
|---|---|---|
| **Name** | Geben Sie eine beschreibende Kennung ein (z. B. „Kesseldruckabfall-Warnung"). | ✅ Ja |
| **Zähler** | Wählen Sie den Zielzähler aus dem Dropdown-Menü. | ✅ Ja |
| **Bedingung** | Wählen Sie die mathematische Logik (z. B. ist größer als, ist kleiner als). | ✅ Ja |
| **Spike-Modell aktivieren** | *(Bedingt)* Erscheint nur, wenn die Bedingung **steigt um** oder **sinkt um** ist. | Optional |
| **Schwellenwert** | Geben Sie den numerischen Wert ein, der den Auslöser aktiviert. | ✅ Ja |
| **Ist aktiv** | Aktivieren Sie dieses Kontrollkästchen, um den Auslöser sofort zu aktivieren. | — |
| **Beschreibung** | Geben Sie Kontext oder Betriebsnotizen an. | — |

## Schritt 4: Abschließen und aktivieren
Klicken Sie auf **Einreichen**, um die Konfiguration zu speichern.
