# Wie man einen Zeitauslöser dupliziert

1. Navigieren Sie zu **Wartung > Zeitauslöser**.
2. Lokalisieren Sie den Auslöser. Klicken Sie auf **Mehr (⋯)** in der rechten Spalte.
3. Klicken Sie auf **Duplizieren**.
4. Das Formular öffnet sich mit der exakten Konfiguration des ursprünglichen Auslösers vorausgefüllt.
   - **Aktualisieren Sie den Namen** sofort, um eine eindeutige Kennung zu gewährleisten.
   - Passen Sie Wiederholung, Start-/Enddatum oder Aktivstatus nach Bedarf an.
5. Klicken Sie auf **Einreichen**.

:::note
Beim Duplizieren wird nur die **Zeitplankonfiguration** kopiert. Der Ausführungsverlauf des ursprünglichen Auslösers wird **nicht** kopiert.
:::
