# Wie man einen Zählerauslöser dupliziert

1. Navigieren Sie zu **Wartung > Zählerauslöser**.
2. Lokalisieren Sie den Auslöser. Klicken Sie auf das **Mehr (⋯)**-Symbol in der rechten Spalte.
3. Klicken Sie auf **Duplizieren** im Dropdown-Menü.
4. Das Erstellungsformular öffnet sich, vorausgefüllt mit den Details des ursprünglichen Auslösers.

:::note
Die Duplizierfunktion wirkt als Klon. Dies ermöglicht Ihnen, Konsistenz ohne manuelle Eingabe zu wahren.
:::

Klicken Sie auf **Einreichen**, um den neuen Auslöser zu speichern.
