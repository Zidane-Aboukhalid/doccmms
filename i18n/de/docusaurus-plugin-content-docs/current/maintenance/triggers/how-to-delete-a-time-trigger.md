# Wie man einen Zeitauslöser löscht

1. Navigieren Sie zu **Wartung > Zeitauslöser**.
2. Lokalisieren Sie den Auslöser. Klicken Sie auf **Mehr (⋯)** in der rechten Spalte.
3. Klicken Sie auf **Löschen** im Dropdown-Menü.

:::warning
Das Löschen eines Zeitauslösers ist eine **dauerhafte Aktion** und kann nicht rückgängig gemacht werden. Wenn Sie die Ausführung des Zeitplans nur vorübergehend stoppen möchten, bearbeiten Sie den Auslöser und **deaktivieren Sie das Kontrollkästchen Ist aktiv**.
:::

4. Bestätigen Sie die Aktion im Bestätigungsdialog.
