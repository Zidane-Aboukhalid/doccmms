# Wie man einen Zählerauslöser löscht

1. Navigieren Sie zu **Wartung > Zählerauslöser**.
2. Lokalisieren Sie den Auslöser. Klicken Sie auf **Mehr (⋯)** in der letzten Spalte.
3. Wählen Sie die Option **Löschen** (rot hervorgehoben).

:::warning
Das Löschen eines Auslösers ist **dauerhaft**. Wenn Sie die Automatisierung nur vorübergehend stoppen möchten, deaktivieren Sie stattdessen das Kontrollkästchen **Ist aktiv** im Bearbeitungsformular.
:::

4. Bestätigen Sie die Aktion im Bestätigungsdialog.
