# Wie man einen Zählerauslöser bearbeitet

1. Navigieren Sie zu **Wartung > Zählerauslöser**.
2. Lokalisieren Sie den Auslöser. Klicken Sie auf **Mehr (⋮)** und wählen Sie **Bearbeiten**.
3. Aktualisieren Sie die erforderlichen Felder:

| Feld | Anweisung | Pflichtfeld |
|---|---|---|
| **Name** | Geben Sie eine klare beschreibende Kennung ein. | ✅ Ja |
| **Bedingung** | Wählen Sie die mathematische Logik. Siehe Tabelle unter [Über Auslöser](./about). | ✅ Ja |
| **Schwellenwert** | Geben Sie den numerischen Wert oder das Intervall ein, das den Auslöser aktiviert. | ✅ Ja |
| **Ist aktiv** | Aktivieren, um sofort zu aktivieren. | — |
| **Beschreibung** | Geben Sie Kontext oder Notizen an. | — |

4. Klicken Sie auf **Einreichen**. Das System wendet sofort die aktualisierte Logik an.
