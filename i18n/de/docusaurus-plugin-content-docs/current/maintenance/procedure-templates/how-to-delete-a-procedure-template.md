# Wie man eine Verfahrensvorlage löscht

1. Navigieren Sie zu **Wartung > Verfahrensvorlagen**.
2. Lokalisieren Sie das Verfahren. Klicken Sie auf **Mehr (⋯)** in der rechten Spalte.
3. Klicken Sie auf die Option **Löschen** (Papierkorb-Symbol).

:::danger Kritische Warnung
Das Löschen einer Verfahrensvorlage ist eine **dauerhafte Aktion**.
- **Zukünftiger Einfluss**: Diese Vorlage steht nicht mehr zum Anhängen an neue Arbeitsaufträge zur Verfügung.
- **Historische Daten**: Das Löschen der Mastervorlage **löscht nicht** die Daten aus bereits abgeschlossenen Arbeitsaufträgen, die dieses Verfahren verwendet haben.
:::

4. Bestätigen Sie die Aktion im Bestätigungsdialog.
