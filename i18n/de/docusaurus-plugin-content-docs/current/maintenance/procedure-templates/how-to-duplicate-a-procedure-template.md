# Wie man eine Verfahrensvorlage dupliziert

1. Navigieren Sie zu **Wartung > Verfahrensvorlagen**.
2. Lokalisieren Sie die Vorlage. Klicken Sie auf **Mehr (⋯)** in der rechten Spalte.
3. Klicken Sie auf **Duplizieren**.
4. Der Builder öffnet sich mit einer **exakten Kopie** der ursprünglichen Konfiguration.

:::warning Entscheidender erster Schritt
**Aktualisieren Sie sofort das Namensfeld**. Wenn Sie es identisch mit dem Original lassen, wissen Techniker nicht, welche SAA sie auswählen sollen.
:::

5. Passen Sie Parameter, Aufgaben oder Logik nach Bedarf an.
6. Überprüfen Sie in **VORSCHAU**, dann klicken Sie auf **Einreichen**.
