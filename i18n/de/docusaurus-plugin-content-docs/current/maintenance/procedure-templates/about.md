# Verfahrensvorlagen

Verfahrensvorlagen ermöglichen Ihnen die Erstellung von **Standardarbeitsanweisungen (SAA)**, Sicherheitschecklisten oder Wartungs-Workflows, die an Arbeitsaufträge angehängt werden können. Sie führen Techniker Schritt für Schritt und erfassen strukturierte Daten.

## Referenzleitfaden: Aufgaben-Eingabetypen

| Eingabetyp | Beschreibung | Bester Anwendungsfall |
|---|---|---|
| **Textfeld** | Ermöglicht dem Techniker eine kurze Textantwort einzugeben. | Seriennummern, Teilebezeichnungen oder Beobachtungsnotizen aufzeichnen. |
| **Kontrollkästchen** | Einfacher Umschalter zur Bestätigung eines einzelnen Elements. | Grundlegende Sichtprüfungen (z. B. „Schutzvorrichtungen vorhanden", „Bereich gereinigt"). |
| **Numerisches Feld** | Beschränkt die Eingabe strikt auf numerische Werte. | Zählerablesungen, Temperaturen, PSI oder Spannungswerte erfassen. |
| **Einzelauswahl** | Präsentiert eine benutzerdefinierte Optionsliste und zwingt den Techniker, genau eine Antwort auszuwählen. | Statusbedingungen (z. B. „Bestanden / Reparatur erforderlich / Nicht bestanden"). |
| **Checkliste** | Präsentiert eine benutzerdefinierte Optionsliste und ermöglicht mehrere Antworten. | Mehrteilige Prüfungen (z. B. „Alle abgeschlossenen Flüssigkeiten auswählen"). |
| **Inspektionskontrolle** | Standardisiertes Zustandseingabeformat für Asset-Inspektionen. | Periodische Audits des physischen Zustands von Assets. |
| **Ja-Nein-N/A** | Bietet drei voreingestellte Schnellauswahl-Schaltflächen. | Standard-Compliance- oder Sicherheitsfragen. |
| **Datum** | Öffnet eine Kalenderauswahl zur Aufzeichnung eines bestimmten Tages. | Ablauf- oder Installationsdaten von Teilen aufzeichnen. |
| **Datum und Uhrzeit** | Öffnet eine Kalender- und Uhrzeitauswahl. | Den genauen Zeitpunkt eines Vorfalls oder Tests aufzeichnen. |
