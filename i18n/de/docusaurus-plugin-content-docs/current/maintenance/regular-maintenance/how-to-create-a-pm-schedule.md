# Wie man einen regulären Wartungsplan (PM) erstellt

## Schritt 1: Zu regulärer Wartung navigieren
Erweitern Sie **Wartung** in der Seitenleiste und wählen Sie **Reguläre Wartungen**. Klicken Sie auf **+ Erstellen**.

## Schritt 2: Allgemeine Parameter definieren

| Feld | Anweisung | Pflichtfeld |
|---|---|---|
| **Name** | Geben Sie eine klare Kennung ein (z. B. „Luftkompressor — Monatliche Inspektion"). | ✅ Ja |
| **Arbeiten an** | Wechseln Sie zwischen Assets oder Anlage und suchen Sie das Ziel. | ✅ Ja |
| **Kategorie** | Kategorisieren Sie die Art der Arbeit. | ✅ Ja |
| **Priorität** | Wählen Sie die Dringlichkeitsstufe. | ✅ Ja |
| **Typen** | Geben Sie den genauen Wartungstyp an. | ✅ Ja |
| **Zuweisen an** | Techniker, der für wiederkehrende Aufgaben verantwortlich ist. | — |
| **Verfahrensvorlage** | Automatisch an jeden generierten Arbeitsauftrag anhängen. | — |

## Schritt 3: Auslöserquelle konfigurieren
Verknüpfen Sie einen **Zeitauslöser** oder einen **Zählerauslöser**, um zu definieren, wann Arbeitsaufträge automatisch generiert werden.

## Schritt 4: Planungsfenster festlegen *(Optional)*

| Feld | Anweisung |
|---|---|
| **Startdatum** | Das Datum, an dem der PM-Plan aktiv wird. |
| **Enddatum** | Das Ablaufdatum. Leer lassen für einen offenen Plan. |

## Schritt 5: Speichern
Klicken Sie auf **Einreichen**, um den regulären Wartungsplan zu aktivieren.
