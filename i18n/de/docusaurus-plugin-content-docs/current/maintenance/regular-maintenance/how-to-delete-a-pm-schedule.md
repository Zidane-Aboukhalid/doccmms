# Wie man einen regulären Wartungsplan löscht

1. Navigieren Sie zu **Wartung > Reguläre Wartungen**.
2. Lokalisieren Sie den Plan. Klicken Sie auf **Mehr (⋯)** in der rechten Spalte.
3. Klicken Sie auf **Löschen** (Papierkorb-Symbol).

:::warning Vor dem Löschen
Wenn Sie den Plan nur **pausieren** möchten, bearbeiten Sie ihn und setzen Sie ein vergangenes **Enddatum**, um die zukünftige Generierung von Arbeitsaufträgen zu stoppen, ohne den Datensatz zu löschen.
:::

:::danger
Das Löschen eines regulären Wartungsplans ist eine **dauerhafte Aktion**. Das System wird dauerhaft aufhören, Arbeitsaufträge aus diesem Plan zu generieren. Bereits generierte Arbeitsaufträge sind **nicht betroffen**.
:::

4. Bestätigen Sie die Aktion im Bestätigungsdialog.
