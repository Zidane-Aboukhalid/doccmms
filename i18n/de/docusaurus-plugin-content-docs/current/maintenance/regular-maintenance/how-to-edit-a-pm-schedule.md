# Wie man einen regulären Wartungsplan bearbeitet

1. Navigieren Sie zu **Wartung > Reguläre Wartungen**.
2. Lokalisieren Sie den Plan. Klicken Sie auf **Mehr (⋯)** und wählen Sie **Bearbeiten**.
3. Aktualisieren Sie die erforderlichen Felder: Allgemeine Parameter, Zuweisung, Auslöserquelle, Verfahrensvorlage, Planungsfenster.
4. Klicken Sie auf **Einreichen**, um Ihre Änderungen anzuwenden.

:::info Betriebliche Notiz
Die Bearbeitung eines regulären Wartungsplans aktualisiert den **Masterplan**. Dies betrifft alle **zukünftigen** Arbeitsaufträge, die aus diesem Plan generiert werden. Bereits generierte oder abgeschlossene Arbeitsaufträge werden **nicht rückwirkend geändert**.
:::
