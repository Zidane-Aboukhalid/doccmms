# Wie man eine Arbeitsanforderung ablehnt

> **Erforderliche Berechtigung:** Arbeitsanforderung ablehnen

1. Navigieren Sie zu **Wartung > Arbeitsanforderungen**.
2. Klicken Sie auf **Mehr (⋯)** in der rechten Spalte der spezifischen Anforderung.
3. Klicken Sie auf **Ablehnen** im Dropdown-Menü.
4. **Grund angeben** *(falls angefordert)*: Das System fragt möglicherweise nach einem Ablehnungsgrund, der für den Antragsteller sichtbar ist.
5. Bestätigen Sie die Aktion. Der Status wechselt zu **Abgelehnt** und der Antragsteller wird benachrichtigt.
