# Wie man eine Arbeitsanforderung löscht

> **Erforderliche Berechtigung:** Arbeitsanforderung löschen (normalerweise Administrator)

:::warning
Bevor Sie löschen, überlegen Sie, ob Sie einen historischen Datensatz behalten müssen. Wenn Sie sie nur aus Ihrer aktiven Warteschlange entfernen möchten, ist **Ablehnen** (was den Datensatz behält) geeigneter als Löschen.
:::

1. Navigieren Sie zu **Wartung > Arbeitsanforderungen**.
2. Klicken Sie auf **Mehr (⋯)** in der rechten Spalte.
3. Klicken Sie auf **Löschen** (Papierkorb-Symbol).
4. Bestätigen Sie die Aktion im Dialog.
