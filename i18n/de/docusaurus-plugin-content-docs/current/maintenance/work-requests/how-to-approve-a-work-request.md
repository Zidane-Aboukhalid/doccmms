# Wie man eine Arbeitsanforderung genehmigt (Konvertierung)

Wenn Sie eine Arbeitsanforderung **genehmigen**, konvertiert das System sie automatisch in einen aktiven **Arbeitsauftrag**.

> **Erforderliche Berechtigung:** Arbeitsanforderung genehmigen (normalerweise Wartungsverantwortlicher oder Administrator)

1. Navigieren Sie zu **Wartung > Arbeitsanforderungen**.
2. Lokalisieren Sie die Anforderung. Klicken Sie auf **Mehr (⋯)** in der rechten Spalte.
3. Klicken Sie auf **Genehmigen**.
4. Das System startet sofort das Formular **Arbeitsauftrag erstellen**, vorausgefüllt mit den Anforderungsdaten.
   - Überprüfen und vervollständigen Sie alle Felder (Kategorie, Priorität, Typen, Zuweisung, Daten).
5. Klicken Sie auf **Einreichen**. Der Status der Anforderung wechselt zu **Genehmigt** und der Arbeitsauftrag wird erstellt.
