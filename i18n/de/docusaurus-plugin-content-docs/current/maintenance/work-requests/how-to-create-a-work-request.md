# Wie man eine Arbeitsanforderung erstellt

## Schritt 1: Zu Arbeitsanforderungen navigieren
Erweitern Sie **Wartung** in der Seitenleiste und wählen Sie **Arbeitsanforderungen**. Klicken Sie auf **+ Erstellen**.

## Schritt 2: Das Problem detaillieren

| Feld | Anweisung | Pflichtfeld |
|---|---|---|
| **Name** | Geben Sie einen klaren, kurzen Titel ein (z. B. „Starke Vibration und Ölleck an Hauptlinie"). | ✅ Ja |
| **Priorität** | Wählen Sie die wahrgenommene Dringlichkeitsstufe. | ✅ Ja |
| **Asset** | Suchen und wählen Sie die fehlerhafte Ausrüstung. | — |
| **Beschreibung** | Geben Sie vollständige Diagnosedetails an. | — |
| **Notiz** | Fügen Sie administrativen Kontext oder Anweisungen für den Prüfer hinzu. | — |

:::tip Best Practice: Asset immer markieren
Obwohl das Asset-Feld technisch optional ist, sollten Sie immer versuchen, Ihre Anforderung mit einer bestimmten Maschine zu verknüpfen, um Verzögerungen im Genehmigungs- und Reparaturprozess zu vermeiden.
:::

## Schritt 3: Arbeitsanforderung einreichen
Klicken Sie auf **Einreichen**, um die Anforderung an die Warteschlange des Verantwortlichen zu senden.
