# Wie man eine Arbeitsanforderung bearbeitet

1. Navigieren Sie zu **Wartung > Arbeitsanforderungen**.
2. Lokalisieren Sie die Anforderung. Klicken Sie darauf oder auf **Mehr (⋯)** und wählen Sie **Bearbeiten**.
3. Aktualisieren Sie die relevanten Felder: **Name**, **Priorität**, **Asset**, **Beschreibung**, **Notiz**.
4. Klicken Sie auf **Einreichen**, um die aktualisierte Arbeitsanforderung zu speichern.
