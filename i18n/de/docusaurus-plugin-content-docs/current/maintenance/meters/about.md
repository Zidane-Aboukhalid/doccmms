# Zähler

## Über Zähler

Das **Zähler**-Modul dient als historisches Verfolgungswerkzeug zur Aufzeichnung physischer Zustände oder Betriebsindikatoren im Laufe der Zeit. Es ermöglicht Wartungsteams, kontinuierliche numerische Daten basierend auf einer definierten Maßeinheit zu erfassen (z. B. Grad Celsius, Kilometer, PSI).

### Funktionsweise

Ein Zähler erstellt eine **Datenzeitlinie** für jede überwachte Variable. Wenn Techniker routinemäßige Kontrollen durchführen, erfassen sie neue numerische Ablesungen zu verschiedenen Zeitpunkten.

**Beispiel:**
> Sie erstellen einen Zähler zur Verfolgung der kritischen Temperatur eines Kühlraums.
> - **Montag**: Ein Techniker erfasst eine normale Basisablesung von **-18 °C**.
> - **Donnerstag**: Eine routinemäßige Inspektion ergibt eine neue Ablesung von **-5 °C**.
>
> Durch einen Blick auf diese einfache Zeitlinie kann der Verantwortliche sofort den Temperaturanstieg erkennen und das Kühlsystem untersuchen, bevor der Bestand beschädigt wird.

### Betrieblicher Nutzen

Durch das kontinuierliche Hinzufügen neuer Ablesungen zum Zähler erstellen Sie einen präzisen, nachvollziehbaren Verlauf des überwachten Zustands. Dies ermöglicht Wartungsteams:

- Betriebsindikatoren in Echtzeit über die gesamte Anlage hinweg zu überwachen.
- **Anomale Trends**, Spitzen oder Einbrüche schnell zu erkennen, die auf ein aufkommendes Betriebsproblem hinweisen könnten.
