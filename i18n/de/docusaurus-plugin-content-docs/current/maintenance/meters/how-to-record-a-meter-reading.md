## Wie man eine Zählerablesung erfasst

> **Erforderliche Berechtigung:** Zählerablesungen verwalten

Es gibt **zwei primäre Methoden** zur manuellen Eingabe einer neuen Ablesung für einen vorhandenen Zähler.

### Methode 1: Schnelles Hinzufügen über die Listenansicht
1. Navigieren Sie zu **Wartung > Zähler**.
2. Lokalisieren Sie den spezifischen Zähler in der Liste.
3. Klicken Sie auf das **Ablesungs**-Symbol in der Zeile dieses Zählers.

### Methode 2: Hinzufügen über die Detailansicht
1. Navigieren Sie zu **Wartung > Zähler**.
2. Klicken Sie auf den Zähler, um seine Detailansicht zu öffnen.
3. Klicken Sie auf die Schaltfläche **Ablesung** in der Oberfläche.

### Daten erfassen

| Feld | Anweisung | Pflichtfeld |
|---|---|---|
| **Zähler** | Das System legt dieses Feld basierend auf Ihrer Auswahl vorher fest. Es kann nicht geändert werden. | ✅ System |
| **Ablesewert** | Geben Sie die aktuellen numerischen Daten des Geräts ein. | ✅ Ja |
| **Notiz** | Erfassen Sie visuelle Beobachtungen oder Kontext zur Ablesung. | — |

Klicken Sie auf **Einreichen**, um die neue Ablesung dem historischen Protokoll des Zählers hinzuzufügen.

:::note
Für die Bearbeitung erfasster Ablesungen ist die Berechtigung **Rückdatierte Zählerhistorienbearbeitung erlauben** erforderlich.
:::
