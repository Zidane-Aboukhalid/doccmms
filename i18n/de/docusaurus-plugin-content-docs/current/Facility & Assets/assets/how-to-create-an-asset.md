title: Vermögenswert erstellen
---

Das Erstellen eines präzisen Asset-Profils ist ein grundlegender Schritt zur Verfolgung von Wartungsaufzeichnungen, zur Berechnung von Ausfallkosten und zur Verwaltung des Lebenszyklus von Geräten.

### Schritt 1: Navigieren zum Asset-Dashboard
Erweitern Sie im Hauptseitenmenü **Anlagen und Vermögenswerte** und wählen Sie **Vermögenswerte**. Klicken Sie auf die schwarze Schaltfläche **+ Erstellen** in der oberen rechten Ecke des Bildschirms.

### Schritt 2: Asset-Parameter definieren
Füllen Sie das Erstellungsformular aus. Felder mit einem roten Sternchen sind Pflichtfelder.

| Feld | Anleitung | Erforderlich |
| :--- | :--- | :--- |
| **Name** | Geben Sie eine eindeutige Kennung ein (z. B. „Luftkompressor 01“). | Ja |
| **Status** | Wählen Sie den aktuellen Betriebsstatus (z. B. Aktiv, In Warteschleife). | Ja |
| **Grund** | Geben Sie den Grund für den gewählten Status an. | Ja |
| **Elternteil** | Wechseln Sie zwischen **Anlage** (eigenständig) oder **Vermögenswert** (Unterkomponente). | Nein |
| **Kritikalität** | Definieren Sie die Bedeutung: Niedrig, Mittel oder Hoch. | Nein |
| **Typen** | Kategorisieren Sie den Vermögenswert (z. B. HLK, Förderband). | Nein |
| **Verantwortliche Teams** | Weisen Sie das zuständige Wartungsteam zu. | Nein |
| **Barcode** | Geben oder scannen Sie einen Tracking-Code für die schnelle Identifizierung ein. | Nein |
| **Ausfallkosten** | Geschätzter finanzieller Verlust pro Stunde Stillstand. | Nein |
| **Kaufdatum** | Wählen Sie das Erwerbsdatum der Ausrüstung aus. | Nein |
| **Anschaffungskosten** | Anfängliche finanzielle Anschaffungskosten. | Nein |

### Schritt 3: Asset-Datensatz speichern
Überprüfen Sie alle eingegebenen Parameter und klicken Sie auf die Schaltfläche **Senden** unten rechts im Formular.
