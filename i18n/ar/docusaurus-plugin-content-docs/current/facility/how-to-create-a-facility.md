title: كيفية إنشاء مرفق
---

اتبع هذه الخطوات لتسجيل موقع مادي جديد أو مبنى أو منطقة تشغيلية في قاعدة بيانات النظام.

### الخطوة 1: الانتقال إلى المرافق
من القائمة الجانبية الرئيسية، قم بتوسيع **المرافق والأصول** وحدد **المرافق**. انقر فوق زر **+ إنشاء** الأسود الموجود في الزاوية العلوية اليمنى من لوحة المعلومات.

### الخطوة 2: تحديد معايير المرفق
أكمل نموذج الإنشاء لتسجيل تفاصيل الموقع وتحديد غرضه التشغيلي.
(ملاحظة: الحقول المميزة بنجمة حمراء إلزامية).

| الحقل | التعليمات | مطلوب |
| :--- | :--- | :--- |
| **الاسم** | أدخل معرفًا واضحًا وفريدًا للموقع (مثل "المستودع الرئيسي"). | نعم |
| **المرفق الأب** | إنشاء تسلسل هرمي مكاني باختيار موقع أوسع يحتويه. | لا |
| **هل هو مخزن** | حدد هذا المربع لتحويل الموقع إلى مركز مخزون نشط. | لا |
| **تاريخ الإنشاء** | حدد تاريخ بناء المرفق أو تشغيله. | لا |
| **العنوان** | أدخل عنوان الشارع المادي أو الموقع الجغرافي. | لا |
| **الوصف** | تقديم ملاحظات عامة أو تعليمات الوصول إلى الموقع. | لا |

:::info منطق النظام
تفعيل خيار **هل هو مخزن** يسمح للمرفق بتخزين وتدوير قطع الغيار والإمدادات والأصول ماديًا.
:::

### الخطوة 3: حفظ المرفق
راجع المعايير المدخلة. انقر فوق زر **إرسال** لتسجيل المرفق نهائيًا في النظام.
