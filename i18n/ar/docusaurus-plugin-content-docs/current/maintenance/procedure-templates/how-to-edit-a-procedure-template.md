# كيفية تعديل قالب إجراء

اتّبع هذه الخطوات لتحديث إجراء تشغيل معياري موجود أو تعديل مهام قائمة التحقق الخاصة به.

1. انتقل إلى **الصيانة > قوالب الإجراءات**.
2. حدد الإجراء. انقر على أيقونة **المزيد (⋯)** وحدد **تعديل**.
3. يُفتَح المنشئ في وضع **المحرر** مع ملء جميع الحقول ببيانات القالب الحالية. يمكنك تحديث:
   - **الاسم**، **الأصل**، **الأنواع**، **الوصف**
   - الكتل الموجودة (النص، نوع المُدخل، تبديل المطلوب، قواعد المنطق)
   - استخدم شريط الأدوات (**H**، **!**، **+**) لإضافة أقسام أو تحذيرات أو مهام جديدة (حد 20 حقلاً)
4. بدّل إلى **المعاينة** للتحقق من تغييراتك، ثم انقر على **إرسال**.

:::info ملاحظة تشغيلية
تحديث قالب الإجراء **يغيّر المخطط الرئيسي**. يُطبَّق هذا التغيير على أي أوامر عمل **مستقبلية** تستخدم هذا القالب، لكنه **لا يُعدّل بأثر رجعي** أوامر العمل المكتملة مسبقاً.
:::
