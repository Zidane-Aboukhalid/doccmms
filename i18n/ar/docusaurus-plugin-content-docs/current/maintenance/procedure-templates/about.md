# قوالب الإجراءات

تُتيح لك قوالب الإجراءات بناء **إجراءات التشغيل المعيارية (SOPs)**، وقوائم الفحص الأمني، وسير عمل الصيانة التي يمكن إرفاقها بأوامر العمل. توجّه الفنيين خطوة بخطوة وتلتقط البيانات بشكل مُنظَّم.

---

## دليل مرجعي: أنواع مُدخلات المهام

يضمن اختيار نوع المُدخل الصحيح بقاء سجلات صيانتك نظيفة وقابلة للتقرير.

| نوع المُدخل | الوصف | أفضل حالة استخدام |
|---|---|---|
| **حقل نصي** | يسمح للفنّي بكتابة إجابة نصية قصيرة ومفتوحة. | تسجيل الأرقام التسلسلية أو أسماء القطع أو ملاحظات الملاحظة الموجزة. |
| **خانة اختيار** | مربع تبديل بسيط للتأكيد على عنصر واحد. | التحقق البصري الأساسي (مثل: «الحواجز في مكانها»، «المنطقة نُظّفت»). |
| **حقل رقمي** | يقصر الإدخال على القيم الرقمية فقط. | تسجيل قراءات العداد، درجات الحرارة، الضغط، أو مستويات الجهد. |
| **اختيار واحد** | يعرض قائمة مخصصة من الخيارات ويُجبر الفنّي على اختيار إجابة واحدة فقط. | حالات الحالة (مثل: «اجتاز / يحتاج إصلاح / فشل»). |
| **قائمة تحقق** | يعرض قائمة مخصصة من الخيارات ويسمح باختيار إجابات متعددة. | الفحوصات متعددة الأجزاء (مثل: «اختر جميع السوائل المُستكملة: الزيت، مياه التبريد، سائل الفرامل»). |
| **فحص تفتيشي** | تنسيق إدخال حالة موحّد مُصمَّم خصيصاً لعمليات تفتيش الأصول. | التدقيقات الفيزيائية الدورية لحالة الأصول. |
| **نعم-لا-لا ينطبق** | يوفر ثلاثة أزرار اختيار سريع مُعدَّة مسبقاً. أسرع من إنشاء قائمة اختيار واحد مخصصة. | أسئلة الامتثال أو السلامة المعيارية (مثل: «هل زر الإيقاف الطارئ يعمل؟»). |
| **تاريخ** | يفتح منتقي تقويم لتسجيل يوم محدد. | تسجيل تواريخ انتهاء الصلاحية أو تواريخ تركيب القطع. |
| **تاريخ ووقت** | يفتح منتقي تقويم وساعة لتسجيل اليوم والوقت الدقيق. | تسجيل اللحظة الدقيقة التي وقعت فيها مشكلة أو اكتمل فيها اختبار. |
