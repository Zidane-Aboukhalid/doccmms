# كيفية تعديل طلب عمل

قبل قبول طلب العمل أو رفضه، قد تحتاج إلى تحديث تفاصيله.

1. انتقل إلى **الصيانة > طلبات العمل**.
2. حدد طلب العمل المحدد. انقر عليه لفتح عرض التفاصيل، أو انقر على أيقونة **المزيد (⋯)** وحدد **تعديل**.
3. حدّث الحقول ذات الصلة حسب الحاجة:
   - **الاسم**: وضّح عنوان المشكلة.
   - **الأولوية**: اضبط الإلحاح إذا تغيّر الوضع.
   - **الأصل**: حدّث المعدة المتأثرة أو أضفها.
   - **الوصف**: قدّم ملاحظات أو سياقاً أكثر تفصيلاً.
   - **ملاحظة**: أضف تعليمات للمدير المراجع.
4. انقر على زر **إرسال** لحفظ طلب العمل المحدّث.
