# كيفية تعديل جدول صيانة دورية

1. انتقل إلى **الصيانة > الصيانة الدورية**.
2. حدد الجدول المحدد. انقر على أيقونة **المزيد (⋯)** وحدد **تعديل**.
3. حدّث الحقول الضرورية:
   - **المعلمات العامة**: الاسم، الفئة، الأولوية، أو الوصف
   - **التعيين**: إعادة التعيين لفنّي مختلف
   - **مصدر المشغّل**: تغيير المشغّل الزمني أو مشغّل العداد
   - **قالب الإجراء**: إرفاق أو تغيير قائمة التحقق
   - **نافذة الجدول**: ضبط تاريخ البداية أو النهاية
4. انقر على **إرسال** لتطبيق تغييراتك.

:::info ملاحظة تشغيلية
تعديل جدول الصيانة الدورية يحدّث **المخطط الرئيسي**. يؤثر على جميع أوامر العمل التي ستُنشأ **مستقبلاً**. **لا يُعدّل بأثر رجعي** أوامر العمل المنشأة أو المكتملة مسبقاً.
:::
