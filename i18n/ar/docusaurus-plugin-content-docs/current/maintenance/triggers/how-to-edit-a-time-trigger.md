# كيفية تعديل مشغّل زمني

1. انتقل إلى **الصيانة > المشغّل الزمني**.
2. حدد المشغّل في القائمة. انقر على أيقونة **المزيد (⋯)** في العمود الأقصى يميناً وحدد **تعديل**.
3. حدّث الحقول في نموذج تعديل المشغّل الزمني:

| الحقل | التعليمات |
|---|---|
| **الاسم** | حدّث معرّف النص لهذا الجدول. |
| **نشط** | ضع علامة أو أزلها لتفعيل أو تعطيل المشغّل دون حذف الجدول. |
| **التكرار** | عدّل نمط الجدولة ووقت التنفيذ الدقيق. |
| **تاريخ البداية** | اضبط التاريخ الذي يصبح فيه المشغّل نشطاً. |
| **تاريخ النهاية** | اضبط التاريخ الذي ينتهي فيه المشغّل. |
| **الوصف** | حدّث السياق أو الملاحظات التشغيلية. |

4. انقر على **إرسال** لتطبيق تغييراتك.

:::note
إذا عدّلت إعدادات **التكرار**، سيُعيد النظام فوراً حساب الجدول ويحدّث **تاريخ التشغيل التالي** في لوحة التحكم الرئيسية.
:::
