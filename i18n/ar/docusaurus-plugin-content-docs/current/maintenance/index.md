# الصيانة

**قسم الصيانة** هو القلب التشغيلي لـ UniCMMS. يجمع كل الأدوات التي يحتاجها فريقك لإدارة وتتبع وأتمتة جميع أنشطة الصيانة.

---

## الوحدات الفرعية

اختر قسماً لمعرفة المزيد:

*   [**أوامر العمل**](/docs/maintenance/work-orders/about) — إنشاء وتعيين وتتبع مهام الصيانة.
*   [**طلبات العمل**](/docs/maintenance/work-requests/about) — تقديم ومراجعة تذاكر الصيانة.
*   [**الصيانة الدورية (PM)**](/docs/maintenance/regular-maintenance/about) — جدولة خطط الصيانة الوقائية.
*   [**قوالب الإجراءات**](/docs/maintenance/procedure-templates/about) — بناء إجراءات التشغيل وقوائم التحقق.
*   [**العدادات**](/docs/maintenance/meters/about) — تتبع الحالات الفيزيائية والمقاييس التشغيلية.
*   [**المشغّلات**](/docs/maintenance/triggers/about) — أتمتة سير العمل بقواعد مبنية على الشروط.
