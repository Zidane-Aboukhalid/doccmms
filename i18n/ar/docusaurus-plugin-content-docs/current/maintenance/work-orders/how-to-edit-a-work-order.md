# كيفية تعديل أمر عمل

اتّبع هذه الخطوات لتعديل مهمة موجودة، كإعادة تعيين فنّي أو ضبط موعد نهائي أو إضافة مزيد من السياق.

1. انتقل إلى **الصيانة > أوامر العمل**.
2. حدد أمر العمل المحدد. انقر على أيقونة **المزيد (⋯)** في العمود الأقصى يميناً وحدد **تعديل**.
3. سيُفتَح نموذج تعديل أمر العمل مع ملء جميع الحقول ببيانات المهمة الحالية. حدّث الحقول الضرورية.
4. انقر على زر **إرسال** لحفظ أمر العمل المحدّث.
