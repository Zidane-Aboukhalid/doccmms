# أوامر العمل

**أمر العمل** هو المحرّك الرئيسي لعمليات الصيانة لديك. إنه سجل رقمي رسمي وقابل للتتبّع يُجيز مهمة صيانة محددة ويُفصّلها ويُراقبها من البداية إلى النهاية.

سواء كان فريقك يستجيب لعطل مفاجئ في المعدات (**الصيانة التصحيحية**) أو يُنفّذ فحصاً أمنياً روتينياً (**الصيانة الوقائية**)، يعمل أمر العمل بوصفه **المصدر الوحيد للحقيقة**. يُخبر فنييك بالضبط:
- ماذا يجب فعله
- أين يُنجَز
- من المسؤول
- متى يكون موعد استحقاقه

...مع تتبّع **التكلفة المالية** للعمالة والقطع في الوقت ذاته.

---

## دورة حياة أمر العمل وإدارة الحالة

يمثّل أمر العمل عمالة نشطة وإنفاقاً مالياً. لضمان سلامة البيانات، يستخدم النظام **آلة حالة صارمة**. يجب أن يمرّ أمر العمل عبر مراحل دورة حياة محددة، وبعض تغييرات الحالة ستُطلق إجراءات مالية وأمانات دائمة.

### فهم مجموعات الحالة

يوجد كل أمر عمل في أحد حالتين متمايزتين:

| الحالة | الأوضاع | الوصف |
|---|---|---|
| **قيد المعالجة (نشط)** | مسودة، مفتوح، قيد التنفيذ، معلّق، قيد المراجعة | أمر العمل «حيّ». يمكن للفنيين تعديل التفاصيل وتسجيل القطع وضبط الجداول. لا تُخصَم أي قطع بشكل دائم من مخزون المستودع الحيّ بعد. |
| **نهائي (مُغلق)** | مكتمل، ملغى | أمر العمل «مات». يصبح **سجلاً تاريخياً ومالياً مقفلاً**. |

### مصفوفة انتقال الحالة

لا يمكن للمستخدمين تجاوز خطوات سير العمل. يقيّد النظام تحركات الحالة إلى هذه المسارات المنطقية:

| الحالة الحالية | الحالة التالية المسموحة | صلاحيات النظام |
|---|---|---|
| **مسودة** | مفتوح | مسموح: تعديل التفاصيل، تسجيل القطع، ضبط الجداول. |
| **مفتوح** | قيد التنفيذ | مسموح: تعديل التفاصيل، تسجيل القطع، ضبط الجداول. |
| **قيد التنفيذ** | معلّق، قيد المراجعة، مكتمل، أو ملغى | مسموح: تعديل التفاصيل، تسجيل القطع، ضبط الجداول. |
| **معلّق** | قيد التنفيذ، قيد المراجعة، مكتمل، أو ملغى | مسموح: تعديل التفاصيل، تسجيل القطع، ضبط الجداول. |
| **قيد المراجعة** | قيد التنفيذ، مكتمل، أو ملغى | مسموح: تعديل التفاصيل، تسجيل القطع، ضبط الجداول. |
| **مكتمل** | *(مقفل)* | **إجراء النظام**: يُنهي تكاليف العمالة فوراً ويخصم المخزون بشكل دائم. |
| **ملغى** | *(مقفل)* | قراءة فقط. لا يمكن تعديل أي شيء. |

### الضمانات

**القاعدة أ:** لن يسمح النظام بإكمال أمر عمل إذا كانت القطع المستخدمة غير متوفرة في المخزون.

**القاعدة ب:** فور وضع علامة **مكتمل** أو **ملغى**، يصبح الملف الكامل **للقراءة فقط**.

**القاعدة ج:** يمكن لمسؤولي النظام تهيئة تجاوز على مستوى المستأجر يسمح للمستخدمين المخوّلين بإعادة فتح أوامر العمل المُغلقة عبر **بوابة المسؤول > إعدادات المستأجر > صلاحيات التشغيل**.
