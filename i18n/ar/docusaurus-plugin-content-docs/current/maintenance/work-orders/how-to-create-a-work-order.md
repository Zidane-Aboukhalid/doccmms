# كيفية إنشاء أمر عمل

اتّبع هذه الخطوات لإنشاء مهمة قابلة للتنفيذ لفريق الصيانة.

## الخطوة 1: الانتقال إلى أوامر العمل
من الشريط الجانبي الرئيسي، وسّع **الصيانة** وحدد **أوامر العمل**. انقر على زر **+ إنشاء** في الزاوية العلوية اليمنى.

## الخطوة 2: إكمال نموذج الإنشاء

| الحقل | التعليمات | مطلوب |
|---|---|---|
| **الاسم** | أدخل عنواناً واضحاً وموجزاً للمهمة (مثل: «استبدال فلتر الهواء الرئيسي»). | ✅ نعم |
| **الحالة** | اختر الحالة الحالية للأمر (مسودة أو مفتوح). | ✅ نعم |
| **العمل على** | بدّل بين الأصول أو المرافق، ثم ابحث عن الهدف المحدد. | ✅ نعم |
| **الفئة** | صنّف طبيعة العمل (مثل: تصحيحي، وقائي، أو غيره). | ✅ نعم |
| **الأولوية** | اختر مستوى الإلحاح: لا شيء، منخفض، متوسط، عالٍ، أو حرج. | ✅ نعم |
| **الأنواع** | حدد نوع الصيانة الدقيق (مثل: استبدال، سلامة، فحص). | ✅ نعم |
| **التعيين إلى** | اختر المستخدم أو الفنّي المسؤول عن تنفيذ العمل. | — |
| **تاريخ البداية المقترح** | اختر التاريخ والوقت الذي يجب أن يبدأ فيه العمل. | — |
| **تاريخ الاستحقاق** | اختر الموعد النهائي المطلق لإكمال المهمة. | — |
| **المدة المقدّرة** | أدخل وقت العمل المتوقع بالساعات والدقائق. | — |
| **الوصف** | أدخل التفاصيل التشخيصية الكاملة أو التعليمات التشغيلية. | — |
| **حساب الميزانية** | اختر الحساب المالي المنطبق لتتبّع النفقات. | — |
| **قسم الرسوم** | اختر القسم الداخلي المسؤول عن التكلفة. | — |

:::tip تلميح لمسؤول النظام
هل تفتقد خياراً محدداً في **الأنواع** أو **حساب الميزانية** أو **قسم الرسوم**؟ تُتحكَّم هذه القوائم عالمياً. يجب إضافة القيم المفقودة في وحدة **البيانات المرجعية** أولاً.
:::

## الخطوة 3: إرسال أمر العمل
راجع جميع البيانات المُدخَلة للتأكد من دقتها. انقر على زر **إرسال** لإنشاء أمر العمل وإشعار الفنّي المعيّن فوراً.
