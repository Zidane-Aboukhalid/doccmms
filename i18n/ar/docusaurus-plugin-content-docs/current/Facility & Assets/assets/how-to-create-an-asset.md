title: كيفية إنشاء أصل
---

يعد إنشاء ملف تعريف دقيق للأصل خطوة أساسية لتتبع سجلات الصيانة وحساب تكاليف التوقف وإدارة دورات حياة المعدات.

### الخطوة 1: الانتقال إلى لوحة معلومات الأصول
من القائمة الجانبية الرئيسية، قم بتوسيع قائمة **المرافق والأصول** وحدد **الأصول**. انقر فوق زر **+ إنشاء** الأسود الموجود في الزاوية العلوية اليمنى من الشاشة.

### الخطوة 2: تحديد معايير الأصل
أكمل نموذج الإنشاء. الحقول المميزة بنجمة حمراء إلزامية.

| الحقل | التعليمات | مطلوب |
| :--- | :--- | :--- |
| **الاسم** | أدخل معرفًا فريدًا (مثلاً "ضابط هواء 01"). | نعم |
| **الحالة** | حدد الحالة التشغيلية الحالية (مثلاً نشط، قيد الانتظار). | نعم |
| **السبب** | حدد مبرر الحالة المختارة. | نعم |
| **الأب** | التبديل بين **مرفق** (مستقل) أو **أصل** (مكون فرعي). | لا |
| **الأهمية** | تحديد الأهمية التشغيلية: منخفضة، متوسطة، أو عالية. | لا |
| **الأنواع** | تصنيف الأصل (مثلاً HVAC، ناقل). | لا |
| **الفرق المسؤولة** | تعيين فريق الصيانة المسؤول عن هذه المعدات. | لا |

### الخطوة 3: حفظ سجل الأصل
راجع جميع المعايير المدخلة وانقر فوق زر **إرسال** الموجود أسفل يمين النموذج.
