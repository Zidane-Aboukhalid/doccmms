# Cómo eliminar una Plantilla de Procedimiento

1. Vaya a **Mantenimiento > Plantillas de Procedimientos**.
2. Localice el procedimiento. Haga clic en **Más (⋯)** en la columna derecha.
3. Haga clic en la opción **Eliminar** (ícono de papelera).

:::danger Advertencia crítica
Eliminar una Plantilla de Procedimiento es una **acción permanente**.
- **Impacto futuro**: Esta plantilla ya no estará disponible para adjuntarla a nuevas Órdenes de Trabajo.
- **Datos históricos**: Eliminar la plantilla maestra **no borra** los datos de las Órdenes de Trabajo ya completadas con este procedimiento.
:::

4. Confirme la acción en el cuadro de diálogo de confirmación.
