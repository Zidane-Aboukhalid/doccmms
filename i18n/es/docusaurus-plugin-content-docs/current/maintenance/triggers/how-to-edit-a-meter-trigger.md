# Cómo editar un Disparador de Medidor

1. Vaya a **Mantenimiento > Disparadores de Medidor**.
2. Localice el disparador. Haga clic en **Más (⋮)** y seleccione **Editar**.
3. Actualice los campos necesarios:

| Campo | Instrucción | Requerido |
|---|---|---|
| **Nombre** | Introduzca un identificador descriptivo claro. | ✅ Sí |
| **Condición** | Seleccione la lógica matemática. Consulte la tabla en [Acerca de los Disparadores](./about). | ✅ Sí |
| **Umbral** | Introduzca el valor numérico o intervalo que activa el disparador. | ✅ Sí |
| **Está activo** | Marque para activar inmediatamente. | — |
| **Descripción** | Proporcione contexto o notas. | — |

4. Haga clic en **Enviar** para guardar. El sistema aplicará inmediatamente la lógica actualizada.
