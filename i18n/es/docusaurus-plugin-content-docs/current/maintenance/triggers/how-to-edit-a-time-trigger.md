# Cómo editar un Disparador Temporal

1. Vaya a **Mantenimiento > Disparador Temporal**.
2. Localice el disparador. Haga clic en **Más (⋯)** y seleccione **Editar**.
3. Actualice los campos en el formulario:

| Campo | Instrucción |
|---|---|
| **Nombre** | Actualice el identificador de texto. |
| **Está activo** | Marque o desmarque para activar/desactivar sin eliminar la programación. |
| **Recurrencia** | Modifique el patrón de programación y la hora de ejecución exacta. |
| **Fecha de inicio** | Ajuste la fecha en que el disparador se vuelve activo. |
| **Fecha de fin** | Ajuste la fecha en que el disparador expira. |
| **Descripción** | Actualice el contexto o las notas operativas. |

4. Haga clic en **Enviar** para aplicar sus cambios.
