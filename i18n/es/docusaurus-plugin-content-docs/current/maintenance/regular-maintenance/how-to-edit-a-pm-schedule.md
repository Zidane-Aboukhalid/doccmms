# Cómo editar un programa de Mantenimiento Regular

1. Vaya a **Mantenimiento > Mantenimientos Regulares**.
2. Localice el programa. Haga clic en **Más (⋯)** y seleccione **Editar**.
3. Actualice los campos necesarios: Parámetros generales, Asignación, Fuente del Disparador, Plantilla de Procedimiento, Ventana de programación.
4. Haga clic en **Enviar** para aplicar sus cambios.

:::info Nota operativa
Editar un programa de Mantenimiento Regular actualiza el **plan maestro**. Esto afecta a todas las Órdenes de Trabajo **futuras** generadas desde este programa. **No modifica retroactivamente** las Órdenes de Trabajo ya generadas o completadas.
:::
