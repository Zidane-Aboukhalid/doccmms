# Mantenimiento Regular (Mantenimiento Preventivo)

El **Mantenimiento Regular** (PM) le permite programar **tareas de mantenimiento recurrentes y automatizadas** para mantener sus equipos funcionando de manera confiable y prevenir averías inesperadas.

## ¿Por qué usar el Mantenimiento Preventivo?

| Mantenimiento Correctivo | Mantenimiento Preventivo |
|---|---|
| Reactivo — reparar después del fallo | Proactivo — prevenir fallos |
| Tiempo de inactividad no planificado | Tiempo de inactividad planificado y programado |
| Costes de reparación de emergencia más altos | Costes de mantenimiento a largo plazo más bajos |
| El equipo falla de forma impredecible | Vida útil del equipo prolongada |

## Tipos de disparadores PM

### Opción A: Disparador Temporal
El programa PM se activa según un calendario (ej.: cada semana, cada mes).
- En el campo **Disparador**, seleccione un **Disparador Temporal** existente.

:::tip
Los Disparadores Temporales deben crearse en **Mantenimiento > Disparadores** antes de poder vincularlos aquí.
:::

### Opción B: Disparador de Medidor
El programa PM se activa en función de los datos de uso de un Medidor (ej.: cada 500 horas de funcionamiento).
- En el campo **Disparador**, seleccione un **Disparador de Medidor** existente.

:::tip
Los Disparadores de Medidor deben crearse en **Mantenimiento > Disparadores** antes de poder vincularlos aquí.
:::
