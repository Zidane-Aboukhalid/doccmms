# Cómo crear un programa de Mantenimiento Regular (PM)

## Paso 1: Acceder a Mantenimiento Regular
Desde la barra lateral, expanda **Mantenimiento** y seleccione **Mantenimientos Regulares**. Haga clic en **+ Crear**.

## Paso 2: Definir los parámetros generales

| Campo | Instrucción | Requerido |
|---|---|---|
| **Nombre** | Introduzca un identificador claro (ej.: «Compresor Aire — Inspección Mensual»). | ✅ Sí |
| **Trabajar en** | Alterne entre Activos o Instalación y busque el objetivo. | ✅ Sí |
| **Categoría** | Categorice la naturaleza del trabajo. | ✅ Sí |
| **Prioridad** | Seleccione el nivel de urgencia. | ✅ Sí |
| **Tipos** | Especifique el tipo exacto de mantenimiento. | ✅ Sí |
| **Asignar a** | Técnico responsable de las tareas recurrentes. | — |
| **Plantilla de Procedimiento** | Adjuntar automáticamente a cada OT generada. | — |

## Paso 3: Configurar la fuente del Disparador
Vincule un **Disparador Temporal** o un **Disparador de Medidor** para definir cuándo se generan las Órdenes de Trabajo automáticamente.

## Paso 4: Establecer la ventana de programación *(Opcional)*

| Campo | Instrucción |
|---|---|
| **Fecha de inicio** | La fecha en que el programa PM se activa. |
| **Fecha de fin** | La fecha de expiración. Deje en blanco para un plan abierto. |

## Paso 5: Guardar
Haga clic en **Enviar** para activar el programa de Mantenimiento Regular.
