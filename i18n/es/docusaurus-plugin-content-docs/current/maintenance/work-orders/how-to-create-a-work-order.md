# Cómo crear una Orden de Trabajo

## Paso 1: Acceder a Órdenes de Trabajo
Desde la barra lateral, expanda **Mantenimiento** y seleccione **Órdenes de Trabajo**. Haga clic en **+ Crear**.

## Paso 2: Completar el formulario de creación

| Campo | Instrucción | Requerido |
|---|---|---|
| **Nombre** | Introduzca un título claro y conciso (ej.: «Reemplazo del filtro de aire principal»). | ✅ Sí |
| **Estado** | Seleccione el estado actual (Borrador o Abierto). | ✅ Sí |
| **Trabajar en** | Alterne entre Activos o Instalación, luego busque la cible. | ✅ Sí |
| **Categoría** | Categorice la naturaleza del trabajo (ej.: Correctivo, Preventivo). | ✅ Sí |
| **Prioridad** | Seleccione el nivel de urgencia: Ninguna, Baja, Media, Alta, o Crítica. | ✅ Sí |
| **Tipos** | Especifique el tipo exacto de mantenimiento (ej.: Reemplazo, Seguridad, Inspección). | ✅ Sí |
| **Asignar a** | Seleccione el usuario o técnico responsable. | — |
| **Fecha de inicio sugerida** | Seleccione la fecha y hora de inicio. | — |
| **Fecha de vencimiento** | Seleccione la fecha límite absoluta. | — |
| **Duración estimada** | Introduzca el tiempo de trabajo estimado en Horas y Minutos. | — |
| **Descripción** | Introduzca los detalles diagnósticos completos o instrucciones. | — |
| **Cuenta presupuestaria** | Seleccione la cuenta financiera aplicable. | — |
| **Departamento de cargo** | Seleccione el departamento responsable del coste. | — |

## Paso 3: Enviar la Orden de Trabajo
Haga clic en **Enviar** para crear la Orden de Trabajo y notificar inmediatamente al técnico asignado.
