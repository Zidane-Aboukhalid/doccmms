# Cómo duplicar una Orden de Trabajo

1. Vaya a **Mantenimiento > Órdenes de Trabajo**.
2. Localice la Orden de Trabajo. Haga clic en **Más (⋯)** y luego en **Duplicar**.
3. Un nuevo formulario se abre pre-rellenado con los datos exactos de la tarea original.

:::warning Primer paso crucial
Debe actualizar la **Fecha de inicio sugerida** y la **Fecha de vencimiento**. Si deja las fechas históricas, sus métricas de planificación serán incorrectas inmediatamente.
:::

4. Haga clic en **Enviar** para guardar esta nueva Orden de Trabajo distinta.
