# Cómo eliminar una Orden de Trabajo

:::info Restricción del sistema
Solo puede eliminar una Orden de Trabajo si su estado actual es **Borrador** o **Abierto**. Una vez que progresa más allá de estas etapas iniciales, la opción Eliminar queda permanentemente bloqueada.
:::

1. Vaya a **Mantenimiento > Órdenes de Trabajo**.
2. Asegúrese de que el estado sea **Borrador** o **Abierto**, luego haga clic en **Más (⋯)**.
3. Haga clic en **Eliminar**.

:::danger Advertencia: Pistas de auditoría
Si necesita cancelar una tarea pero conservar el registro, use la función **Marcar como** para cambiar el estado a **Cancelado** en lugar de eliminar.
:::

4. Confirme la acción en el cuadro de diálogo de confirmación.
