# Cómo editar una Orden de Trabajo

1. Vaya a **Mantenimiento > Órdenes de Trabajo**.
2. Localice la Orden de Trabajo. Haga clic en **Más (⋯)** y seleccione **Editar**.
3. El formulario se abre pre-rellenado con los datos actuales. Actualice los campos necesarios.
4. Haga clic en **Enviar** para guardar la Orden de Trabajo actualizada.
