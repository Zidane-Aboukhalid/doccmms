# Solicitudes de Trabajo

Una **Solicitud de Trabajo** es una presentación formal para crear una Orden de Trabajo de mantenimiento, reparaciones o problemas de instalaciones.

| Tipo de usuario | Cómo envían |
|---|---|
| **Usuarios internos** | El personal con acceso restringido (Solicitantes) crea Solicitudes de Trabajo directamente desde la aplicación. |
| **Usuarios externos** | Los administradores pueden configurar **Portales de Invitados** web donde personas sin cuenta pueden enviar solicitudes. |

## El proceso de aprobación

Los Responsables de Mantenimiento o Administradores deben revisar todas las solicitudes entrantes para **aprobarlas** o **rechazarlas**.

- **Conversión**: Una vez aprobada, la Solicitud de Trabajo se convierte inmediatamente en una Orden de Trabajo activa.

:::tip Cuándo omitir la Solicitud de Trabajo
Solo los usuarios «Solicitante» con acceso limitado están obligados a usar este flujo. Los **Administradores del sistema y usuarios con acceso completo** pueden crear Órdenes de Trabajo directamente.
:::

## Portales de Invitados

Un **Portal de Invitado** es un formulario web independiente que permite a cualquier persona reportar un problema **sin necesidad de una cuenta**. Cualquiera que tenga la URL dedicada o escanee el código QR del portal puede generar instantáneamente una Solicitud de Trabajo.
