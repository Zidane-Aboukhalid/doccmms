# Cómo editar una Solicitud de Trabajo

1. Vaya a **Mantenimiento > Solicitudes de Trabajo**.
2. Localice la solicitud. Haga clic en ella o en **Más (⋯)** y seleccione **Editar**.
3. Actualice los campos relevantes: **Nombre**, **Prioridad**, **Activo**, **Descripción**, **Nota**.
4. Haga clic en **Enviar** para guardar la Solicitud de Trabajo actualizada.
