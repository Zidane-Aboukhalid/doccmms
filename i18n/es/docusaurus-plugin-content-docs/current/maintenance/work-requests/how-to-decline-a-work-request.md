# Cómo rechazar una Solicitud de Trabajo

> **Permiso requerido:** Rechazar Solicitud de Trabajo

1. Vaya a **Mantenimiento > Solicitudes de Trabajo**.
2. Haga clic en **Más (⋯)** en la columna derecha de la solicitud específica.
3. Haga clic en **Rechazar** en el menú desplegable.
4. **Proporcionar un motivo** *(si se solicita)*: El sistema puede pedirle que introduzca un motivo de rechazo, visible para el solicitante.
5. Confirme la acción. El estado cambiará a **Rechazado** y el solicitante será notificado.
