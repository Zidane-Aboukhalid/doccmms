# Medidores

## Acerca de los Medidores

El módulo de **Medidores** funciona como una herramienta de seguimiento histórico utilizada para registrar estados físicos o indicadores operativos a lo largo del tiempo. Permite a los equipos de mantenimiento registrar datos numéricos continuos basados en una unidad de medida definida (ej.: grados Celsius, kilómetros, PSI).

### Cómo funciona

Un medidor crea una **línea de tiempo de datos** para cualquier variable monitorizada. A medida que los técnicos realizan controles de rutina, registran nuevas lecturas numéricas en diferentes momentos.

**Ejemplo:**
> Crea un medidor para hacer seguimiento de la temperatura crítica de una cámara frigorífica.
> - **Lunes**: Un técnico registra una lectura de referencia normal de **-18 °C**.
> - **Jueves**: Una inspección rutinaria arroja una nueva lectura de **-5 °C**.
>
> Al consultar esta simple línea de tiempo, el responsable puede detectar inmediatamente el aumento de temperatura e investigar el sistema de refrigeración antes de que se dañe el inventario.

### Valor operativo

Al añadir continuamente nuevas lecturas al medidor, construye un historial preciso y rastreable del estado monitorizado. Esto permite a los equipos de mantenimiento:

- Monitorizar indicadores operativos en tiempo real a través de toda la instalación.
- Detectar rápidamente **tendencias anómalas**, picos o caídas que puedan indicar un problema operativo emergente.
