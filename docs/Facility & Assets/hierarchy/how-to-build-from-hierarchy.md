title: How to Build from the Hierarchy View
---

You do not need to navigate away from the tree to expand your infrastructure. You can generate new records directly from the Hierarchy dashboard.

### Step 1: Navigate to Hierarchy
From the main sidebar, expand the **Facilities and Assets** menu and select **Hierarchy**.

### Step 2: Open Create Menu
Click the black **+ Create** button located in the top-right corner of the dashboard.

### Step 3: Select Record Type
A dropdown menu will appear. Select either **Asset** or **Facility** based on what you need to add to the database.

:::info Navigation Note
Selecting an option from this menu will trigger the same creation forms used throughout the system.
*   To see fields required for adding a location, refer to the [How to Create a Facility](/docs/facility/how-to-create-a-facility) section.
*   To see fields required for adding equipment, refer to the [How to Create an Asset](/docs/assets/how-to-create-an-asset) section.
:::
