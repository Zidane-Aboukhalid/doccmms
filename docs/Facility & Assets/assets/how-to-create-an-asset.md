title: How to Create an Asset
---

Creating an accurate asset profile is a fundamental step for tracking maintenance records, calculating downtime costs, and managing equipment lifecycles.

### Step 1: Navigate to Assets Dashboard
From the main sidebar, expand the **Facilities and Assets** menu and select **Assets**. Click the black **+ Create** button located in the top-right corner of the screen.

### Step 2: Define Asset Parameters
Complete the creation form. Fields marked with a red asterisk are mandatory.

| Field | Instruction | Required |
| :--- | :--- | :--- |
| **Name** | Enter a unique identifier (e.g., "Air Compressor 01"). | Yes |
| **Status** | Select current operational state (e.g., Active, On Hold). | Yes |
| **Reason** | Specify the justification for the chosen status. | Yes |
| **Parent** | Toggle between **Facility** (standalone) or **Asset** (sub-component). | No |
| **Criticality** | Define operational importance: Low, Medium, or High. | No |
| **Types** | Categorize the asset (e.g., HVAC, Conveyor). | No |
| **Responsible Teams** | Assign the maintenance team in charge. | No |

### Step 3: Save Asset Record
Review all entered parameters and click the **Submit** button at the bottom-right of the form.
