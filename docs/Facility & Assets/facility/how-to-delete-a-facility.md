title: How to Delete a Facility
---

Follow these steps to permanently remove a physical location or operational zone from the system database.

### Step 1: Navigate to Facilities
From the main sidebar, expand **Facilities and Assets** and select **Facilities**.

### Step 2: Open Action Menu
Identify the facility you wish to remove. Click the **More (⋯)** icon in the last column of the row.

### Step 3: Select Delete
Choose the **Delete** (Trash icon) option.

:::warning System Warning
Deletion is permanent. The system will block this action if the facility is currently serving as a storage location and contains active inventory. You must move or remove all stored parts and supplies before the system allows deletion.
:::

### Step 4: Confirm Deletion
If no preventions are in place, a confirmation prompt will appear. Verify the facility name, then confirm.
