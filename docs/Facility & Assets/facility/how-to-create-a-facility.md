title: How to Create a Facility
---

Follow these steps to register a new physical location, building, or operational zone in the system database.

### Step 1: Navigate to Facilities
From the main sidebar, expand **Facilities and Assets** and select **Facilities**. Click the black **+ Create** button located in the top-right corner of the dashboard.

### Step 2: Define Facility Parameters
Complete the creation form to log the location's details and establish its operational purpose.
(Note: Fields marked with a red asterisk are mandatory).

| Field | Instruction | Required |
| :--- | :--- | :--- |
| **Name** | Enter a clear, unique identifier for the location (e.g., "Main Warehouse"). | Yes |
| **Parent Facility** | Establish a spatial hierarchy by selecting a broader location. | No |
| **Is a Storage** | Check this box to convert the location into an active inventory hub. | No |
| **Created Date** | Select when the facility was constructed or commissioned. | No |
| **Address** | Enter the physical street address or geolocation. | No |
| **Description** | Provide general notes or location access instructions. | No |

:::info System Logic
Enabling **Is a Storage** allows the facility to physically store and rotate spare parts, supplies, and assets.
:::

### Step 3: Save Facility
Review the entered parameters. Click the **Submit** button to permanently register the facility in the system.
