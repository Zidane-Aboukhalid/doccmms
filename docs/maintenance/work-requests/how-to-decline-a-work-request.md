# How to Decline a Work Request

If a maintenance request is not valid, is a duplicate, or cannot be actioned at this time, you can formally decline it.

> **Required Permission:** Decline Work Request (typically Maintenance Manager or Administrator)

1. Navigate to **Maintenance > Work Requests** and locate the Work Request in the dashboard.
2. Click the **More (⋯)** icon in the far-right column of the specific Work Request row.
3. Click the **Decline** option from the dropdown menu.
4. **Provide a Reason** *(if prompted)*: The system may prompt you to enter a reason for declining. This reason is visible to the requester.
5. Confirm the action. The Work Request status will update to **"Declined"** and the requester will be notified.
