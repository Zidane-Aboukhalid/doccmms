# How to Edit a Regular Maintenance Schedule

Follow these steps to modify an existing PM plan.

1. Navigate to **Maintenance > Regular Maintenances**.
2. Locate the specific PM schedule. Click the **More (⋯)** icon in the far-right column and select **Edit**.
3. The Edit form will open, pre-filled with the current schedule's data. Update the necessary fields:
   - **General Parameters**: Name, Category, Priority, or Description
   - **Assignment**: Reassign to a different technician
   - **Trigger Source**: Change or re-link the Time Trigger or Meter Trigger
   - **Procedure Template**: Attach or change the SOP checklist
   - **Schedule Window**: Adjust Start Date or End Date
4. Click the **Submit** button to apply your changes.

:::info Operational Note
Editing a Regular Maintenance schedule updates the **master blueprint**. This affects all **future** auto-generated Work Orders from this schedule. It does **not** retroactively alter previously generated or completed Work Orders.
:::
