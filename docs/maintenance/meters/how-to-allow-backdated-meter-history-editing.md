## How to Allow Backdated Meter History Editing

This permission enables users to manage historical meter data, including **adding past readings**, **editing existing entries**, and **deleting records**. Access to these actions is restricted to users with the appropriate permissions.

When this option is **disabled**, users are restricted to creating new readings only. In this state, the timestamp is automatically set to the current date and time and cannot be modified.

### Enable the Permission
1. Go to **Admin Portal > Tenant Settings > Operational Permissions**.
2. Select **Allow Backdated Meter History Editing**.
3. Once enabled, select **More (⋯)** in the History table to access the **Edit** and **Delete** options.
