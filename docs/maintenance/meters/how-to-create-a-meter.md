## How to Create a Meter

> **Required Permission:** Add Meter

Follow these steps to configure a new meter in the system.

### Step 1: Navigate to Meters
From the main sidebar, navigate to **Maintenance** and select the **Meters** sub-menu.

### Step 2: Open the Creation Form
Click the **Create** button located in the top action bar of the Meters list.

### Step 3: Define Meter Parameters
Complete the input fields in the creation form:

| Field | Instruction | Required |
|---|---|---|
| **Name** | Enter a unique identifier (e.g., "Air Compressor 01 Hours"). | ✅ Yes |
| **Asset** | Select the specific equipment this meter belongs to from the dropdown. | — |
| **Measurement Unit** | Define the unit (e.g., Degree Celsius, Liter, Volt, etc.). | ✅ Yes |
| **Reading Value** | Enter the current or initial state of the meter if the asset is already in service. | ✅ Yes |
| **Description** | Provide additional context or details about the meter's purpose or location. | — |

### Step 4: Save
Click the **Submit** button to finalize the meter configuration.
