## How to Update a Meter

> **Required Permission:** Edit Meter

Follow these steps to modify the configuration of an existing meter.

### Step 1: Locate the Meter
Navigate to **Maintenance > Meters**. Use the search bar or filters to find the meter you wish to modify.

### Step 2: Access the Edit Form
Select a meter from the list. Click the options button (**⋯**) and select **Edit**. You can also find the Edit button at the top right of the Meter Details page.

### Step 3: Modify Parameters

| Field | Instruction | Required |
|---|---|---|
| **Name** | Update the identifier for the meter. | ✅ Yes |
| **Asset** | Reassign the meter to a different asset if necessary. | — |
| **Measurement Unit** | Define the unit (e.g., Degree Celsius, Liter, Volt, etc.). | ✅ Yes |
| **Description** | Update the contextual details or purpose of the meter. | — |

:::tip Enable Meter Unit Updates
To enable meter unit updates, go to **Admin Portal > Tenant Settings > Operational Permissions** and select **Allow Update Meter Unit**. The Measurement Unit field will then become editable in the Edit Meter form.
:::

### Step 4: Finalize Changes
Click the **Submit** button to save the updated configuration.
