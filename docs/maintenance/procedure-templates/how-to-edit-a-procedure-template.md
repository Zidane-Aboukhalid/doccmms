# How to Edit a Procedure Template

Follow these steps to update an existing SOP or modify its checklist tasks.

1. Navigate to **Maintenance > Procedure Templates**.
2. Locate the procedure. Click the **More (⋯)** icon and select **Edit**.
3. The builder opens in **EDITOR** mode pre-filled with the current data. You can update:
   - **Name**, **Asset**, **Types**, **Description**
   - Existing step blocks (text, input type, required toggle, logic rules)
   - Use the toolbar (**H**, **!**, **+**) to add new sections, warnings, or tasks (20-field limit applies)
   - Use the icons on any block to **copy** or **delete** it
4. Toggle to **PREVIEW** to verify your changes, then click **Submit**.

:::info Operational Note
Updating a Procedure Template **alters the master blueprint**. This change applies to any **future** work orders using this template, but does **not retroactively alter previously completed** work orders.
:::
