# How to Duplicate a Procedure Template

Duplicating a template allows you to clone a complex SOP — including all its tasks, input types, and conditional logic.

1. Navigate to **Maintenance > Procedure Templates**.
2. Locate the template. Click the **More (⋯)** icon in the far-right column.
3. Click **Duplicate** from the dropdown menu.
4. The builder will open with an **exact copy** of the original template's configuration.

:::warning Crucial First Step
Immediately **update the Name** field. If you leave it identical to the original, you will create database confusion and technicians will not know which SOP to select.
:::

5. Adjust parameters, modify tasks or logic as needed.
6. Switch to **PREVIEW** mode to verify, then switch back to **EDITOR** and click **Submit**.
