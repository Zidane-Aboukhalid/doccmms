title: How to Edit a Facility
---

Follow these steps to modify the operational details of an existing physical location or to reassign its parent hierarchy.

### Step 1: Navigate to Facilities
From the main sidebar, expand the **Facilities and Assets** menu and select **Facilities**.

### Step 2: Open Action Menu
Identify the specific location you wish to edit. Click the **More (⋯)** icon located in the last column of that row.

### Step 3: Select Edit
Choose the **Edit** option from the dropdown menu. The original creation form will open, pre-populated with the facility's current data.

### Step 4: Update and Save
Modify the necessary fields (e.g., updating the name, or moving it into a new parent facility). Click the **Submit** button to save changes permanently.
